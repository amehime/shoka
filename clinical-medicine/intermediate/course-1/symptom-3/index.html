症状 03 咯血

# 概述

咯血的病理生理简述如下：

  1. 由于各种有害物质刺激
  2. 炎症、结核及肿瘤等侵蚀
  3. 左心房压力急剧升高
  4. 由于凝血功能障碍

量：

  1. ＜100ml/24h 以内为小量咯血
  2. 100～500ml/24h 为中等量咯血
  3. ＞500ml/24h（或一次咯血超过 300ml）为大咯血

# 常见病因

  • 呼吸系统疾病：结核最常见
  • 心血管系统疾病：二狭最特殊 肺静脉破裂出血
  • 凝血和出血功能障碍性疾病
  • 传染病和寄生虫病：钩端螺旋体病、流行性出血热、肺吸虫病。

# 临床表现

# 1️⃣ 咯血伴发热

  1. 肺炎：多见痰中带血或铁锈色痰
  2. 肺脓肿：大量有臭味的脓血痰 + 叩诊浊音
  3. 肺结核：低热、疲乏、盗汗
  4. 肺真菌病：老人、小儿、体弱及长期应用激素
  5. 肺寄生虫病
    • 阿米巴： 吞噬红细胞 痰呈棕褐色 + 阿米巴滋养体
    • 肺吸虫： 吸血 铁锈色痰 + 嗜酸性粒细胞↑ 寄生虫感染的表现
    • 肺棘球蚴病：包虫形成的囊肿
  6. 肺梗死：突然发热、胸痛、咳嗽、咯血
  7. 系统性红斑狼疮：伴关节痛 + 皮疹 + 蛋白尿等

# 2️⃣ 咯血伴胸痛

肺炎、肺栓塞

# 3️⃣ 咯血伴痰多

支气管扩张：反复咯血及 继发厌氧菌感染时 咳出大量的脓臭痰

# 4️⃣ 咯血伴皮下出血

多半是全身血液病表现的一个出血症状。

# 小结

  • 青壮年：肺结核、支气管扩张、二尖瓣狭窄
  • 40 岁以上长期吸烟史：支气管肺癌
  • 儿童慢性咳嗽少量咯血：特发性含铁血黄素沉着症
  • 大量咯血：空洞型肺结核、支扩、肺脓肿
  • 支气管肺癌：痰中带血，持续或间断性

# 咯血与呕血鉴别

咯血呕血
病因支扩、肺结核、肺癌溃疡、肝硬化、胃癌
其它症状喉痒、胸闷、咳嗽上腹不适、恶心、呕吐
出血方式咯出呕出，可为喷射状
颜色鲜红暗红，咖啡渣，有时鲜红

# 练习题

  1. 大量咯血是指每日咯血量为

    • 200ml 以上
    • 300ml 以上
    • 400ml 以上
    • 500ml 以上
    • 600ml 以上

  2. 国内咯血的最常见病因是

    • 支气管扩张症
    • 肺结核
    • 慢性支气管炎
    • 支气管肺癌
    • 风湿性心脏病二尖瓣狭窄

  3. 引起肺结核大咯血的原因是

    • 支气管肺癌
    • 结核侵蚀小血管
    • 结核破坏了肺组织
    • 肺结核空洞，空洞内动脉瘤破裂
    • 肺结核所致毛细血管渗透性增高

    肺结核咯血的机制为

    • 结核病变使毛细血管通透性增高，血液渗出，导致痰中带血或小血块；— E
    • 如病变累及小血管使管壁破溃，则造成中等量咯血；— B
    • 如空洞壁肺动脉分支形成的小动脉瘤破裂，或继发的结核性支气管扩张形成的动静脉瘘破裂，则造成大量咯血，甚至危及生命。— D

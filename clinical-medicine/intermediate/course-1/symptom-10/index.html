症状 10 腹泻

# 概述

腹泻是消化系统疾病常见症状之一。由于各种因素作用于胃肠道，使胃肠道的分泌、消化、吸收及运动等功能出现障碍，导致大便次数增多，性状改变。
该症状也是机体的一种防卫机制，通过腹泻使毒素从体内排出，避免中毒。

# 病因 / 发生机制

  1. 高渗性腹泻
    高渗性食物或药物 甘露醇 ➡️水分入肠腔➡️肠道运动加速➡️腹泻
  2. 渗出性腹泻
    黏膜大量渗出病理性渗出物，如炎症性肠病 克隆病、溃疡性结肠炎 ，呈脓血便伴发热
  3. 分泌性腹泻
    毒素 + 肠道黏膜受体➡️cAMP↑➡️分泌水及电解质（如 霍乱 ）
  4. 运动性腹泻
    肠内毒素、药物及消化不全的中间产物➡️ 胃肠运动 功能异常
  5. 吸收障碍性
    各种原因肠道黏膜受损 慢性肠炎肠黏膜受损；慢性胰腺炎，胰液分泌减少，脂肪吸收障碍 ➡️脂肪酸吸收↓➡️肠腔水、电解质及脂肪酸↑➡️肠运动↑

# 诊断方法

# 腹泻伴发热

疾病特点
急性胃肠炎不洁饮食 + 恶心、呕吐、发热 + 腹痛
细菌性痢疾不洁饮食 + 粘液 脓血便 + 里急后重
病毒性肠炎小儿多 + 蛋黄便 + 轮状病毒（+）
白色念珠菌肠炎一老一小 + 抵抗力弱 + 豆腐渣样便
霍乱上吐下泻 + 米泔水样便，无腹痛

# 粘液脓血便

三大常见病：

  1. 痢疾 不洁饮食史，累及直肠而里急后重
  2. 溃疡性结肠炎 钡餐可见铅管征，自身免疫病没有不洁饮食史
  3. 结肠癌 症状进行性加重

# 腹泻伴腹痛

  1. 克罗恩病
    慢性腹泻 + 脐周疼 好发于回肠末端 + 消瘦 吸收不好 + 脓肿 通常无脓血便 + 瘘管及肠梗阻
  2. 阿米巴痢疾
    慢性腹泻 + 果酱样便 阿米巴原虫做阿米巴运动（变形运动），吞噬红细胞 + 低热 + 便可见阿米巴滋养体

# 腹泻伴包块

  1. 肠结核
    慢性病程 + 结核中毒症状 + 便秘腹泻交替 肠道功能紊乱的表现，非特征性症状 可有腹部包块
  2. 肠道肿瘤
    40 岁以上 + 低热 + 进行性体重减轻 + 潜血（+）+CEA + 腹部包块

# 其他

全身感染性疾病、过敏性紫癜及尿毒症等。

# 练习题

  1. 小肠大部分切除所致的腹泻属于

    • 分泌性
    • 动力性
    • 渗透性
    • 渗出性
    • 吸收不良性

    肠管切除后导致短肠综合征，吸收不良。
    A-D 均需刺激肠壁。

  2. 霍乱弧菌所致的腹泻为

    • 渗透性腹泻
    • 吸收不良性腹泻
    • 分泌性腹泻
    • 渗出性腹泻
    • 动力性腹泻

    霍乱弧菌产生毒素刺激肠壁，分泌大量水和电解质。

  3. 伴重度失水的腹泻是下列哪种疾病

    • 霍乱
    • 肠结核
    • 血吸虫病
    • 吸收不良综合征
    • 慢性非特异性溃疡性结肠炎

    A - 毒素刺激肠壁分泌大量水 + 电解质，米泔水样大便
    B - 结核中毒症状，腹部包块，便秘腹泻交替
    C - 不常见
    E - 黏液脓血便

  4. 下列哪种疾病常引起便秘与腹泻交替出现

    • 肠结核
    • 慢性菌痢
    • 阿米巴痢疾
    • 肠易激综合征
    • 慢性非特异性溃疡性结肠炎

    便秘与腹泻交替为肠道功能紊乱表现，见于肠结核、肠道肿瘤
    C - 果酱样大便
    D - 三种便秘，便秘或腹泻或便秘腹泻交替，但不如 A 常见
    E - 黏液脓血便

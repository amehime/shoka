# жҰӮеҝө

е‘•иЎҖе’Ңй»‘дҫҝжҳҜдёҠж¶ҲеҢ–йҒ“пјҲеұҲж°Ҹйҹ§еёҰд»Ҙиҝ‘пјүеҮәиЎҖзҡ„ дё»иҰҒ з—ҮзҠ¶ йқһзү№еҫҒжҖ§ пјҲеӣһзӣІйғЁд»ҘиҝңдёәдёӢж¶ҲеҢ–йҒ“пјҢдёӨиҖ…д№Ӣй—ҙдёәдёӯж¶ҲеҢ–йҒ“пјү


еұҲж°Ҹйҹ§еёҰеҸҲз§° Treitz йҹ§еёҰпјҢд№ҹеҸ«еҚҒдәҢжҢҮиӮ жӮ¬йҹ§еёҰгҖӮ
жӯЈеёёжғ…еҶөдёӢпјҢдёҠдёӢж¶ҲеҢ–йҒ“жҳҜд»ҘеҚҒдәҢжҢҮиӮ з©әиӮ жӣІдёәеҲҶз•Ңзәҝзҡ„гҖӮеҚҒдәҢжҢҮиӮ гҖҒз©әиӮ иў«иҝҷж ·зҡ„з”ұе°‘йҮҸе№іж»‘иӮҢзәӨз»ҙе’Ңз»“зј”з»„з»Үе…ұеҗҢжһ„жҲҗзҡ„еҚҒдәҢжҢҮиӮ жӮ¬йҹ§еёҰеӣәе®ҡдәҺи…№еҗҺеЈҒпјҢе®ғеғҸдёҖжқЎз»ізҙўпјҢе°Ҷе°ҸиӮ жҸҗиө·е№¶еӣәе®ҡеңЁи…№еҗҺеЈҒпјҢз§°дёәеұҲж°Ҹйҹ§еёҰгҖӮ

з”ұдәҺжӯӨйҹ§еёҰзҡ„еӯҳеңЁпјҢдҪҝз©әгҖҒеӣһиӮ зҡ„еҶ…е®№зү©дёҚжҳ“иҝ”жөҒе…ҘеҚҒдәҢжҢҮиӮ жҲ–иғғи…”гҖӮж•ҙдёӘж¶ҲеҢ–йҒ“еҸҜеҲҶдёәдёүдёӘйғЁеҲҶпјҢеҚідёҠж¶ҲеҢ–йҒ“пјҢдёӯж¶ҲеҢ–йҒ“е’ҢдёӢж¶ҲеҢ–йҒ“гҖӮйҖҡеёёд»ҘеҚҒдәҢжҢҮиӮ еұҲж°Ҹйҹ§еёҰдёҺеӣһзӣІйғЁдёәж Үеҝ—иҖҢеҲ’еҲҶгҖӮеұҲж°Ҹйҹ§еёҰд»ҘдёҠзҡ„йғЁеҲҶз§°дёҠж¶ҲеҢ–йҒ“пјҢеҢ…жӢ¬еҸЈи…”гҖҒе’ҪгҖҒйЈҹз®ЎгҖҒиғғгҖҒеҚҒдәҢжҢҮиӮ пјҢдёҠж®өз©әиӮ пјӣдёӢж¶ҲеҢ–йҒ“дёәеӣһзӣІйғЁд»ҘдёӢзҡ„йғЁеҲҶпјҢеҢ…жӢ¬йҳ‘е°ҫгҖҒзӣІиӮ гҖҒз»“иӮ гҖҒзӣҙиӮ пјӣдёӯж¶ҲеҢ–йҒ“дҪҚдәҺеұҲж°Ҹйҹ§еёҰдёҺеӣһзӣІйғЁд№Ӣй—ҙпјҢеҢ…жӢ¬дёӢж®өз©әиӮ гҖҒеӣһиӮ гҖӮ

д»ҘдёҠеҲҶж®өеңЁж¶ҲеҢ–йҒ“еҮәиЎҖеҲҶзұ»дёӯжңүйҮҚиҰҒж„Ҹд№үгҖӮдёҠж¶ҲеҢ–йҒ“жәғз–ЎжҢҮзҡ„е°ұжҳҜеұҲж°Ҹйҹ§еёҰд»ҘдёҠйғЁеҲҶжүҖеҮәзҺ°зҡ„жәғз–ЎпјҢйҖҡеёёжҢҮиғғгҖҒеҚҒдәҢжҢҮиӮ зҡ„жәғз–ЎгҖӮ

  • дёҠж¶ҲеҢ–йҒ“еҮәиЎҖ
    еҮәиЎҖйҮҸеӨ§гҖҒйҖҹеәҰеҝ«пјҢд№ҹеҸҜеҮәзҺ°зәўиүІеӨ§дҫҝ
  • дёӢж¶ҲеҢ–йҒ“еҮәиЎҖ
    еҒңз•ҷж—¶й—ҙй•ҝпјҢд№ҹеҸҜеҮәзҺ°й»‘дҫҝ

еҶіе®ҡдёҠж¶ҲеҢ–йҒ“еҮәиЎҖдёҙеәҠиЎЁзҺ°зҡ„жңҖе…ій”®еӣ зҙ жҳҜеҮәиЎҖйҮҸзҡ„еӨҡе°‘пјҢиҖҢдёҚжҳҜйғЁдҪҚпјҒ

# з—…еӣ 

еёёи§Ғзҡ„з—…еӣ дҫқж¬Ўдёәж¶ҲеҢ–жҖ§жәғз–ЎпјҢйЈҹз®ЎжҲ–иғғеә•йқҷи„үжӣІеј з ҙиЈӮпјҢжҖҘжҖ§иғғй»ҸиҶңз—…еҸҳ е–қй…’гҖҒNAIDS пјҢиғғзҷҢгҖӮ

# еҮәиЎҖйҮҸзҡ„дј°и®Ў

дёҙиЎЁжҲ–жЈҖжҹҘз»“жһңдј°и®ЎеҮәиЎҖйҮҸ
еӨ§дҫҝйҡҗиЎҖиҜ•йӘҢйҳіжҖ§5ml д»ҘдёҠ дёҚеҗҢд№Ұе’ҢжҢҮеҚ—жңүе·®еҲ«пјҢжӯӨдёәеёёз”Ёж ҮеҮҶ
й»‘дҫҝ50ml д»ҘдёҠ 5Г—10
е‘•иЎҖиғғеҶ…и“„з§ҜиЎҖйҮҸиҫҫ 250ml
еӨҙжҳҸгҖҒзңјиҠұгҖҒеҸЈе№ІгҖҒд№ҸеҠӣгҖҒзҡ®иӮӨиӢҚзҷҪгҖҒеҝғжӮёгҖҒеҮәеҶ·жұ—пјҢз”ҡиҮіжҳҸеҖ’дёҖж¬Ўиҫҫ 400ml д»ҘдёҠ ж— еҒҝзҢ®иЎҖзҡ„дёҙз•Ң400ml
е‘ЁеӣҙеҫӘзҺҜиЎ°з«ӯ дј‘е…Ӣ1000ml д»ҘдёҠпјҢжҲ–еҮәиЎҖ > 20% е…Ёиә«иЎҖйҮҸзәҰдёӨеӨ§з“¶еҸҜд№җпјҢ2.5LГ—2=5000mlпјҢ20%еҚі1000ml

  • дҪ“жё©пјҡиҪ»еәҰеҚҮй«ҳ еқҸжӯ»зү©иҙЁзҡ„еҗёж”¶зғӯ пјҢдёҚи¶…иҝҮ 37.5в„ғ
  • иЎҖиұЎпјҡ 3пҪһ4 е°Ҹж—¶еҗҺ еҸҜеҮәзҺ°иЎҖзәўиӣӢзҷҪгҖҒзәўз»ҶиғһвҶ“ еҮәиЎҖжңҖејҖе§ӢиЎҖж¶ІеӨ„дәҺжө“зј©зҠ¶жҖҒпјҢеҲқж¬ЎжүҖжҹҘиЎҖеёёи§„еҸҜдёҚе…·еҸӮиҖғж„Ҹд№ү
  • жҳҜеҗҰжңүжҢҒз»ӯеҮәиЎҖзҡ„еҲӨж–ӯпјҡи§Ғеҗ„и®ә

# дјҙйҡҸз—ҮзҠ¶

  1. е‘•иЎҖдјҙдёӯдёҠи…№з—ӣ
    е‘•иЎҖз¬¬дёҖеӨ§еҺҹеӣ ж¶ҲеҢ–жҖ§жәғз–ЎпјҢдёӯдёҠи…№з—ӣеӨҡдёәиғғйғЁз—…еҸҳпјҢиғғйғЁз—…еҸҳдёүжӯҘжӣІпјҡиғғзӮҺвҮЁиғғжәғз–ЎвҮЁиғғзҷҢ

    • ж¶ҲеҢ–жҖ§жәғз–Ў
      ж…ўжҖ§гҖҒе‘ЁжңҹжҖ§гҖҒиҠӮеҫӢжҖ§дёӯдёҠи…№з—ӣ

      иғғжәғз–ЎвҮЁйӨҗеҗҺз—ӣ
      еҚҒдәҢжҢҮиӮ жәғз–ЎвҮЁйҘҘйҘҝз—ӣ

    • иғғзӮҺ
      йқһиҠӮеҫӢжҖ§дёӯдёҠи…№й’қз—ӣжҲ–йҡҗз—ӣ зЁӢеәҰдёҚеҰӮжәғз–ЎпјҢеҒ¶е°”з—ӣдёҖдёӢ
    • иғғзҷҢ
      дёӯиҖҒе№ҙеӨҡи§ҒпјҢеҸҜеҸҚеӨҚе‘•иЎҖдјҙиҝӣиЎҢжҖ§ж¶ҲзҳҰпјҢжҢҒз»ӯжҖ§жҪңиЎҖйҳіжҖ§

  2. е‘•иЎҖдјҙй»„з–ё
    еЈ¶и…№е‘ЁеӣҙзҷҢ
    еҚҒдәҢжҢҮиӮ д№іеӨҙдҪҚдәҺеЈ¶и…№йғЁпјҢиғҶз®ЎгҖҒиғ°з®Ўе…ұеҗҢејҖеҸЈдәҺеҚҒдәҢжҢҮиӮ д№іеӨҙпјҢжӯӨеӨ„й•ҝиӮҝзҳӨпјҢе өеЎһиғҶз®ЎеҜјиҮҙиғҶжұҒж·Өз§ҜгҖҒй»„з–ё

  3. е‘•иЎҖдјҙиӮқи„ҫеӨ§
    е‘•иЎҖз¬¬дәҢеӨ§еҺҹеӣ йЈҹз®Ўиғғеә•йқҷи„үжӣІеј з ҙиЈӮпјҢд№ҷиӮқвҮЁиӮқзЎ¬еҢ–вҮЁиӮқзҷҢдёүжӯҘжӣІ

    • иӮқзЎ¬еҢ–
      иӮқз—…гҖҒиңҳиӣӣз—ЈгҖҒиӮқжҺҢгҖҒй»„з–ёгҖҒи…№еЈҒйқҷи„үжӣІеј гҖҒи…№ж°ҙ дә§з”ҹжңәеҲ¶пјҡиӮқеҠҹиғҪејӮеёёвҮЁдҪҺиӣӢзҷҪиЎҖз—ҮпјҢй—Ёи„үй«ҳеҺӢвҮЁйқҷж°ҙеҺӢеҚҮй«ҳпјҲиӮқзЎ¬еҢ–и…№ж°ҙеҺҹеӣ д»ҘжӯӨдёәдё»пјү гҖҒи„ҫеӨ§
    • иӮқзҷҢ
      иӮқз—…еҸІгҖҒи…№йғЁжүӘеҸҠиӮҝеқ—гҖҒAFP еўһй«ҳ

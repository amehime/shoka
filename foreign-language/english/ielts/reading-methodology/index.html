# Part 1. 技法精讲

# 雅思阅读中的 “忌讳”

  1. 忌 “陷入文章中，无法自拔”。即不要过度深入地理解文章中的每个单词和句子，而是应该关注文章的主要内容和结构。
  2. 忌 “翻译式阅读”。即不要将英文原文翻译成中文再进行理解，这样会严重影响阅读速度和准确性。
  3. 忌 “纠结于某个生词”。即不要在一个生词上花费过多的时间，而是应该尝试猜测词义或者跳过该词。
  4. 忌 “无目的阅读”。即不要在没有明确目的和问题的情况下进行阅读，而是应该先看问题，再有针对性地阅读文章。
  5. 忌 “忽略题目要求”。即不要忽略题目要求中的关键词或限制条件，否则可能会导致误解或错误回答问题。

# 雅思真经阅读法

# 雅思阅读文章的特点

  1. 文章比较长，是杂志类文章。
  2. 不同于传统的阅读方法，雅思的文章结构和段落分布有其独特的特点。
  3. 题目通常考察文章中的 12 个重要信息点。
  4. 选中心思想时，可以通过阅读题干来抓住文章大意。
  5. 正确率与做题速度可以两条路线来达到，一是通过练习提高正确率，二是每次都追求 100% 确定答案。
  6. 对于不确定的答案，可以通过多找一组同义替换来确认。
  7. 动物类文章考察的特点都是为了动物的生存。

# 快速阅读技巧

  1. 扩大视距

    • 把阅读材料多远离自己约 5 厘米，让眼睛得到舒适的调节距离
    • 扩大眼睛的扫描范围，每次可以接收更多信息
    • 训练眼睛一次接收多个词语或整个短语，而不是一个词一个词接收

  2. 跳读

    • 重点读懂眼睛扫过的名词、动词等重要单词
    • 对不认识或不重要的词语快速略过，不要停顿或反复
    • 根据自身词汇量，把阅读材料划分为重点词语和非重点词语
    • 例如 “the history of civilization is in town of the way we have learned to manipulate water resources” 这句话，重点读懂 “history”、“civilization”、“water"，其他词语快速略过

  3. 变速

    • 根据对词语或短语的理解程度，采用不同的阅读速度
    • 读懂的内容读得慢一些，完整接收信息
    • 不懂的内容快速略过，不要死磕每个词语
    • 避免匀速阅读，正确采用变速技巧可以节省大量时间

  4. 文章的最后一段和第二段最重要

# 读题方法

  1. 慢速读题，每题读 2 遍

    • 第 1 遍读清题意，但不必完全掌握
    • 第 2 遍针对不清楚的部分进行重点理解

  2. 用自己记忆的方法改写题意

    • 使用比较简单的词语改写题目核心内容
    • 辅助记忆，接近日常表达

  3. 画 keywords 关键词前必须读明白题目

    • 否则关键词容易画错，找原文也会出错
    • 一定先读清题意，再代表题意的关键词

# 题干中三类 vocabulary

  1. 主题词

    • 即使不认识题干中的主题词，也不用太担心
    • 因为文章中会反复出现并解释这个主题词，可以根据上下文推测
    • 例如原文中 “pagoda” 这个主题词

  2. 模板词

    • 例如 description、reference、statement 等模板词
    • 不同题干中这些词会反复出现，但本身没有实际意义
    • recognize 这些模板词，不要在意题干中是否出现
    • 建议提前背掉

  3. 考点词

    • 必须重点掌握的词汇考点，例如 “resemble”
    • 这些词汇在真题中反复被考察，要掌握它们的同义词替换
    • 例如原文中 “resemble” 及其同义词 “like” 反复出现在相关题干中

# 三类 keywords 定位

  1. 眼球级词汇

    • 数字、符号、格式醒目的词汇
    • 容易吸引眼球的词汇，一目了然

  2. 朴素级词汇

    • 简单实意的词汇，尤其是名词和动词
    • 记忆简单，直观表达意思
    • 例如匹配题中的 “foot” 和 “hand”

  3. 逻辑关系词

    • 表示并列、转折、因果等逻辑关系的词语
    • 出现在题干中，原文中也必然有对应表示
    • 例如并列词 “and”

# 段中找两点

  • 在自然段中找出两组 heading 同义替换，并找到其原型，从而确定正确的两点。

  • 这种方法需要重点阅读整个自然段，而不是仅仅阅读首句、二句或末句。

  • 如果找到两组词的同义替换，就可以确定一条正确的主线，正确率可以达到 90% 以上。

  • 如果找到三组词的同义替换，则可以确定正确答案，正确率达到 100%。

  • 这种方法可以帮助提高精细的阅读理解能力，而不是简单地刷题。

# 找并列结构

  • 首先，关注并列关系关键词，例如 “and”、“or”、“as well as” 等。这些关键词可以帮助我们识别并列关系的句子结构。
  • 其次，注意并列结构的前后内容，通常它们会有相似或相反的逻辑关系。
  • 最后，通过题干中出现的并列结构，在文章中搜索相应的关键词或句子结构，以确定答案所在的位置。
  • 通过以上方法，可以在文章中快速找到并列结构，进而帮助我们更好地理解文章内容和解题。

# 快速阅读和对比阅读提高做题正确率

  • 要提高做题正确率，需要通过快速阅读和对比阅读来训练。
  • 强调的是快速阅读和对比阅读，不强调精读。
  • 什么是精读呢？比如每句话不认识，单词查出来，这句话读不懂语法分析一下，长难句分析一下。天天这么练，一篇文章花了一个小时，终于把它精读完了，但是永远没有练。
  • 对比阅读，是找出所有的替换，然后画出来，而且一定要在卷子上画出来，把答案中圈儿标出来。每道题都这么做一下，留下做题思路。十年之后或者 20 年之后再来翻，就知道当时是怎么做的。
  • 对于同义替换要敏感，要练要有这个意识，特别强调的这个意识。
  • 首先要把 400 多个考点词背了，背单词，是要敏感，要敏感，对同义替换要极度敏感，要练要有这个意识。
  1. 如何画关键词：
    在阅读文章时，可以通过以下方法画出关键词：
    • 关注名词和动词：名词和动词是句子中的重要成分，它们通常包含了文章中的主要信息。可以根据上下文理解，标记出重要的名词和动词。
    • 关注转折词：转折词如 “但是”、“然而” 等，通常表示前后两个句子之间有对比或矛盾关系。通过关注这些关键词，可以更好地理解文章内容。
    • 关注重复出现的词汇：重复出现的词汇往往是文章的重点，通过画出这些关键词，可以更好地把握文章的主旨。
    • 关注段落中心句：段落中心句通常概括了该段落的中心内容，可以通过画出这些句子，快速把握段落大意。
  2. 如何快速阅读：
    快速阅读的关键是避免逐字逐句阅读，而是采用以下方法：
    • 关注文章结构：在阅读文章时，可以首先关注文章的结构，了解文章的组织方式和主要内容。
    • 扫读：可以采用扫读的方式，快速浏览文章，把握文章的大意和重要信息。
    • 略读：可以略读一些细节信息，如例子、解释等，以加快阅读速度。
    • 阅读题干：在阅读文章之前，可以先浏览题干，了解题目考查的内容和要求。这样可以有目的地去阅读文章，提高阅读效率。
  3. 如何进行同义替换：
    在英语阅读中，同义替换是一种非常常见的方法，可以帮助学生更好地理解文章内容。以下是一些方法：
    • 了解常用同义词：英语中有很多同义词，了解这些同义词的含义和使用方法，可以帮助学生在阅读时更好地理解文章内容。
    • 注意同义词的语境：虽然有些单词在语义上相似，但它们在不同的语境中可能会有不同的含义。因此，学生需要了解单词在不同语境中的用法和含义。
    • 练习同义替换：可以在平时的阅读练习中，尝试用同义词替换文章中的词汇，以加深对文章的理解。
    • 注意同义替换的准确性：在进行同义替换时，需要确保替换的词汇在语义和用法上与原词汇一致，以避免误解文章内容。

# 如何提高雅思阅读的正确率

  1. 第一条路：每次做题都卡时间，但是正确率很低，然后不断的通过练习提升正确率。
  2. 第二条路：保证每次的正确率最高，每道题都看明白，找出所有的同义替换，然后选最确定的答案。如果拿不准，就多找一组同义词，再比较几句话，最终答案就会浮出水面。

视频中强调了正确率优先，说明了方法好了，正确率才能提升。不用背新词或学新的语法，只需在现有的阅读水平上调整方法，达到最高的正确率。

# 总纲标注法

  • 在每道题的题干中画出关键词
  • 把每道题选项中的每个词在原文中找到对应处标注，增加做题的准确率。
  • 这样可以充分展现文本和题目之间的内在联系。

# 乱序查找信息题和理解题

雅思阅读中的乱序查找信息题和理解题是两种常见的题型，它们主要考察考生的阅读能力和理解能力。

乱序查找信息题通常会给出一个问题，要求考生在文章中找到与问题相关的信息。这些信息在文章中的排列可能是无序的，因此考生需要仔细阅读全文，找出相关信息，并理解它们之间的联系。这种题型考察的是考生的阅读速度和信息检索能力。

理解题则是要求考生根据文章内容理解并回答问题。这些问题可能涉及到文章的主旨、细节、推理和判断等方面。考生需要理解文章内容，把握文章的结构和逻辑关系，并能够运用相关的知识和技能来解决问题。这种题型考察的是考生的阅读理解能力、推理能力和判断能力。

因此，读题和理解题目是非常重要的。考生需要先读懂题目，明确问题的内容和要求，然后再回到文章中找到相关的信息和理解文章内容。在阅读文章时，考生需要注意文章的关键词、句子和段落，把握文章的主旨和重要信息。同时，考生还需要注意文章的语言特点，如语法、词汇和表达方式等，这有助于加深对文章的理解。

总之，在雅思阅读中，读题和理解题目是解决乱序查找信息题和理解题的关键。通过加强阅读训练和理解能力的提升，考生可以提高阅读速度和理解力，更好地应对各种阅读题型。

# 匹配题的解题技巧

匹配题通常包括段落匹配题、多项匹配题和 “NB” 题等。这些题型都需要根据阅读材料的内容，找到与题目相关的信息并进行匹配。

  • 解题技巧：

    1. 慢速读题读两遍，把题目都读明白了，不用画都可以找得到。
    2. 画 keywords 的目的是为了记忆这道题，因此画的词一定要是提重心词。
    3. 第一题读两遍读明白，知道它讲什么，然后迅速回原文中找印象深刻的词。

  • 重点说明：

    1. 中心句：在阅读材料中，每个段落都会有中心句，用来表达该段落的中心意思。可以通过寻找中心句的方式来快速定位与题目相关的信息。中心句通常出现在段首或段尾，但也有可能在段中。
    2. 细节信息：匹配题通常需要匹配的是一些具体的细节信息，而不是整个段落。因此，在寻找相关信息时，需要注意细节，找到与题目相关的具体信息点。
    3. 警惕性：在做匹配题时需要保持警惕，避免被相似选项或误导信息所迷惑。特别是对于 “NB” 题，需要仔细比较选项和文章内容，排除掉错误的选项。
    4. 盲人看：在做匹配题时，需要像盲人一样 “看”，即不要被无关的内容所干扰，专注于寻找与题目相关的信息。
    5. 复习笔记：在备考过程中，可以制作复习笔记，将常见的单词和短语记录下来，帮助记忆和理解。
    6. 统一替换：在解题过程中，如果遇到相同的单词或短语，可以考虑进行统一替换，避免混淆。

  • 注意事项：

    1. 如果是 true/false/not given 题，一般来说很少匹配第一题。
    2. 重视读题，读完第一题才能选出正确答案。
    3. 匹配题中，有可能给出的这句话是某个自然段的中心句或中心思想。
    4. 乱序是第三个让人感到痛苦的点，因为好不容易在 a 段的前面找到一句话，感觉这句话肯定就是第二题，但后面还有那么多段，不能放松警惕，因为一旦 NB，这个字还要让学生选一个句子。
    5. 有时候某个自然段可能没有出题。

  • “NB” 题
    是指在进行句子匹配的时候，可能会遇到乱序的情况，即题目中给出的信息与原文中的句子顺序不一致。这种情况需要特别警惕，不能放松警惕，因为一旦出现 “NB”，可能需要选择多个句子才能完成题目。

    • 针对 “NB” 题，可以采取以下应对策略：
      1. 仔细阅读题目中的信息，确定需要匹配的句子范围。
      2. 回到原文中，找到与题目中信息相关的自然段，仔细阅读。
      3. 注意自然段的中心句或中心思想，以及细节信息，这些都可能是题目中需要匹配的句子。
      4. 如果遇到乱序的情况，需要多加留意，不能放松警惕，确保选到正确的句子。
      5. 对于不确定的题目，可以先标记，等全部题目做完后再回来思考。
    • 总之，应对 “NB” 题需要仔细阅读题目和原文，确定需要匹配的句子范围，注意自然段的中心句或中心思想以及细节信息，遇到乱序的情况需要多加留意，不能放松警惕。通过不断的练习和积累经验，可以逐渐提高应对 “NB” 题的能力。

# Part 2. 句子解题

# 单词题

Summary, SC, SA, Table, Flowchart

# summary 填空题

  • 做 summary 填空题时，需要找到空格前后的重要信息并在原文中寻找同义替换。具体步骤如下：

    1. 通读全文：先快速浏览全文，了解文章的大意和结构。
    2. 仔细阅读：仔细阅读空格前后的句子，找出其中的关键词和短语。
    3. 寻找同义替换：在原文中寻找与空格前后的关键词和短语意义相同的表达方式。这可能需要再次阅读同一句子或段落的不同部分，或者跳读其他段落。
    4. 确定答案：根据找到的同义替换，确定空格的正确答案。
    5. 检查答案：检查所填答案是否符合原文的意思，是否通顺、合理。

    总之，做 summary 填空题需要仔细阅读、寻找同义替换和理解原文。

  • 通过排除法解决雅思阅读中遇到的生词。具体方法如下：

    1. 首先，需要仔细阅读题目，了解问题的方向和类型，再阅读选项。
    2. 然后，根据题目中的关键词，在文章中定位相关信息。如果选项中的某些单词在原文中没有出现，就可以直接排除掉。
    3. 最后，如果选项中出现了原文中的相关信息，但是与题目不相关，也可以排除掉。

    通过排除法，可以快速地找到正确答案，提高阅读效率。同时，在阅读过程中，也可以积累一些常见的同义词、反义词、形近词等，帮助理解文章中的生词。

  • 可以通过上下文猜测生词词义的方法。具体步骤如下：

    1. 在原文中找到含有生词的句子，并确定该句子所在的上下文，即原文中附近的几行文字；
    2. 寻找上下文中与生词相关的同义词或近义词，这些词语可能是生词的替代词或相关词；
    3. 根据上下文的语境和逻辑关系，推断出生词的词义。

    需要注意的是，猜测生词词义并不是万能的，如果上下文中的信息不足或逻辑关系不明确，该方法可能无法准确猜测出生词的词义。因此，在考试中，最好还是依靠平时积累的词汇量和语言能力来尽可能准确地理解原文。

# 句子题

TFNG/Multiple Choice

  • 雅思阅读中多选题的处理方法是要定位并列结构。不是整篇文章都需要精读，而是需要通过定位关键词，找到并列结构，从而更好地理解题目和原文之间的关系。

  • 解答雅思题目中的多选题，以下是一些技巧：

    1. 阅读题目并确定关键词。在多选题中，通常会有一个主题或关键词，帮助你理解题目主要询问的是什么方面的内容。通过找到关键词，你可以更好地理解题目并确定正确的答案。
    2. 仔细阅读每个选项，并确定它们与主题或关键词的关系。对于每个选项，问问自己它是否与题目相关，是否符合题目的要求。有时，可能需要阅读整个文章或段落才能确定某个选项是否正确。
    3. 在官方给出的答案中，寻找与题目相关的正确信息。在官方答案中，通常会给出多个正确答案，需要仔细阅读每个选项并确定它们是否符合题目的要求。需要注意的是，正确答案的数量可能会根据题目的不同而有所不同。
    4. 如果你不确定某个选项是否正确，可以再次阅读题目和选项，并尝试找到更多的相关信息来支持你的选择。同时，也可以尝试排除一些明显错误的选项，缩小正确答案的范围。

  • 在官方给出的答案中寻找正确信息的重要性在于：这些答案是经过专业人士仔细考虑和审查后得出的，因此具有很高的准确性和可靠性。通过仔细阅读官方答案，你可以更好地理解题目并确定正确的答案，从而在考试中取得更好的成绩。

  • 填空题技巧：

    1. 阅读题目和给出的句子，了解需要填入的内容。
    2. 仔细阅读相关段落，寻找需要填空的信息。
    3. 确保填入的信息与题目要求一致，特别是注意语法和拼写。
    4. 如果遇到需要填入数字的题目，需要特别注意数字的准确性和单位的正确使用。

  • True/False/Not Given 题技巧：

    1. 阅读题目和选项，了解需要判断哪些陈述是正确的、错误的或者没有提到的。
    2. 仔细阅读相关段落，寻找与选项相关的信息。
    3. 如果遇到需要判断某个陈述是否与文章一致的题目，需要注意文章中的细节和措辞。
    4. 对于那些答案是 False 或 Not Given 的选项，需要有足够的证据来支持你的判断，而不是仅仅依靠猜测。
    5. 在回答问题时，要遵循题目要求，对于不确定的选项，不要随意猜测，而是标记为待定，回头再做考虑。

  • 变词性填空题是雅思阅读中比较常见的一种题型，要求根据句子的结构或者语法规则，将一个词转换成指定的词性进行填空。以下是一些应对这种题型的技巧：

    1. 确定需要填空的词性。在填空前，先确定所填入的词需要转换成什么词性，例如动词、名词、形容词等。
    2. 寻找合适的转换方式。根据需要填空的词性和句子结构，寻找合适的转换方式，例如动词转变成名词形式、名词转变成形容词形式等。
    3. 注意语法规则和拼写。在进行变词性填空时，需要遵循相关的语法规则和拼写规则，以确保填入的内容与题目要求一致。
    4. 练习和提高语感。变词性填空需要一定的语感和语法基础，因此可以通过多读、多写来提高语感和语法水平，从而更好地应对这种题型。
      总之，在应对变词性填空题时，需要先确定需要填空的词性，然后寻找合适的转换方式，并遵循相关的语法规则和拼写规则。同时，需要通过多读、多写来提高语感和语法水平，以更好地应对这种题型。

# 出题者的心态

  1. 出题者的心态
    出题者需要保持客观公正的心态，不能随意改写或添加原文内容。在出题时，需要从剑桥的思路出发，深入理解剑桥的命题思路和考点设置，从而保证题目符合剑桥的要求。
  2. 出题的三种题型
    从逻辑上来说，出题只有三种命题形式，即针对一句话、针对上下文和针对不相邻文章的总结。针对一句话的出题方式，需要选出正确选项对原文进行改写；针对上下文的出题方式，需要归纳总结原文中的多句话，选出正确选项与原文意思相符；针对不相邻文章的出题方式，需要总结原文中的多篇文章，选出正确选项与原文意思相符。

  • 针对不同题型有不同的解题方法和注意事项。

    • 对于单项选择题，需要注意题干中的关键词和选项之间的差异，从原文中找到相关证据后再做出选择；
    • 对于多项选择题，需要注意选项之间的重叠和差异，理解原文意思后再做出选择；
    • 对于判断题，需要注意原文中的细节和关键词，理解原文意思后再进行判断；
    • 对于填空题，需要根据题干中的关键词，从原文中找到相关证据后进行填空。

    总之，在解题时需要注意题目类型和难度，根据不同题型采用不同的解题方法，同时还需要注意阅读速度和准确性的平衡，以及注意时间分配和策略运用。

  • 出题的三种模式：

    1. 针对原文一句话出题，选出正确答案需要对原文进行同义词改写。
    2. 针对原文相邻多句话出题，选出正确答案需要对上下文进行归纳。
    3. 针对原文不相邻的句子出题，选出正确答案需要对不同部分的文章进行总结和改写。

  • 如何应对单选题：
    需要理解出题的三种模式，掌握相应的解题技巧。
    对于针对原文一句话出题的题目，需要寻找与原文相关的同义词或替换表达。
    对于针对原文相邻多句话出题的题目，需要理解上下文的逻辑和关系。
    对于针对原文不相邻的句子出题的题目，需要理解不同部分文章之间的联系和总结。
    同时，还需要注意题目中的关键词和陷阱，以避免错误的选择。

  • 通过排除法选出正确选项需要注意以下几点：

    1. 理解题意：在选择之前，首先要确保自己充分理解题目的意思，明确题目要求和选项的内容。
    2. 分析选项：在理解题意的基础上，对每个选项进行分析，找出其中的错误或不准确之处。可以通过对比不同选项之间的差异，或是考虑一些常识或逻辑来判断。
    3. 考虑极端情况：有时候，排除法的最好方式是考虑极端情况。例如，如果四个选项分别是 A、B、C 和 D，你可以考虑如果某个选项是正确的，那么其他选项一定是错误的。这样可以更容易地排除错误选项。

  • 利用触题命题方式来出题需要注意以下几点：

    1. 确定主题：首先需要确定所要考察的主题或知识点，这有助于你设计题目和选项。
    2. 设计迷惑选项：在出题时，可以设计一些迷惑选项，这些选项可能看起来很合理，但实际上是错误的。这样可以增加题目的难度和区分度。
    3. 确保题目公平性：在设计题目时，需要确保题目对于所有学生都是公平的。这意味着题目不应该依赖于学生已经掌握的特定知识或背景信息，而应该基于题目中给出的明确信息。
    4. 测试题目：在题目发布之前，你需要先测试一下题目，以确保其正确性和公平性。这可以帮助你发现并修正任何潜在的问题或错误。

  • 雅思阅读的正确选项类型通常包括以下几种：

    1. 直接证据：这种选项直接引用了原文中的句子或段落，提供了明确的证据来支持答案。
    2. 间接证据：这种选项没有直接引用原文中的句子或段落，而是通过概括、归纳或推断的方式得出答案。
    3. 字面理解：这种选项涉及到对单词或短语的字面理解，通常是通过简单的同义替换来得出答案。
    4. 重组信息：这种选项将原文中的信息进行重新组合，形成新的答案。

  • 通过画关键词来找到答案是一种有效的阅读技巧。以下是一些关于如何画关键词的建议：

    1. 画出题目中的关键词：在阅读文章之前，先画出题目中的关键词，这有助于你快速理解题目要求，并且在阅读文章时更容易找到答案。
    2. 画出重要概念：画出文章中的重要概念，例如专业术语、数字、地名等。这有助于你更好地理解文章内容，并且可以在做题时快速找到相关信息。
    3. 画出逻辑关系：画出文章中的逻辑关系，例如因果关系、转折关系等。这有助于你理解文章的逻辑结构，并且可以在做题时快速找到相关联的信息。
    4. 画出时间、地点等限定词：画出文章中的时间、地点等限定词，这有助于你快速定位相关信息，并且可以在做题时避免混淆。
      需要注意的是，画关键词只是帮助你快速找到答案的一种技巧，最终的答案还需要根据题目要求进行判断和选择。

# Part 3. 段落理解和匹配题

# Matching 题型和 Heading 题型

  • Matching 题型和 Heading 题型是雅思阅读中的两种不同题型。

  • Matching 题型是一种乱序的题型，需要考生根据题目中的线索去匹配相关信息。这种题型比较讨厌，因为它需要考生在文章中找出相关信息并进行匹配，而且题目之间没有固定的顺序。但是，如果考生能够掌握这种题型的做题方法，优先做细节题，那么这种题型就会变得相对容易一些。

  • Heading 题型是一种选择题，要求考生根据文章内容选择一个最能够概括该段内容的标题。这种题型需要对全文有一定的理解，因为每个自然段都包含新的信息，而这些信息很可能就是中心句或中心思想。一般来说，一篇文章可能会考到全文的每个章节的大意，同时再考一些细节的句子或单词，以此来全面考察考生对文章的理解。

  • 在做这两种题型时，考生需要注意它们的做题方法和顺序。除了 Matching 和 Heading 之外，其他的全部都是顺序的题，只有 Matching 和 Heading 是乱序的。

  • 因此，在做题时需要先做顺序的题，然后再做乱序的题。

# 三种阅读考题的组合方式:

  1. 还我漂亮权 (heading 打头的文章)：优先做细节题，再选择中心思想；

  2. 面目全非 (matching 打头的文章)：优先做后面的题型；

  3. 职权 (没有 heading 和 matching 的文章)：从第一种题型开始做。

不同的文章和题型组合，有不同的最佳应对方法。不能用一种固定的阅读法。

# 做题与学习技巧

  • 做题技巧:

    1. 并列结构猜词：看不懂一个单词，可以根据它前后的词判断它的意思。
    2. 关系型 not given 和 FALSE 的区分：
      如果 a 和 b 都出现但没提及两者关系，则选 not given;
      如果关系出现但与题目描述不符，则选 FALSE。
    3. 剑桥真题中，一个段落只对应一种题型，不会前后交叉。

  • 学习技巧:

    1. 不要一套套做题，而要分类练习不同题型的文章，找到规律。
    2. 学习单词背后的用法，而不仅是记忆词义，达到学以致用的效果。

# Part 4. 真题组合技巧

# 匹配题 (matching)

  • matching 题分为三种:

    1. which program contains 信息型，难度最大
    2. 关系型 matching, 难度中等，考察人名、动物、树木等与其特征之间的关系匹配
    3. 句子 matching, 难度较低，考察句子的上下匹配

  • 做题顺序：先做关系型 matching, 再做句子 matching, 最后做 which program contains 信息型

  • 做题方法:

    • 关系型 matching: 关键在于把所有题干都读懂，不能仅仅依赖人名找答案，而要对题目有全面的理解
    • 句子 matching: 不要浏览所有选项，直接根据题干快速在原文中定位答案，然后在选项中找出匹配内容

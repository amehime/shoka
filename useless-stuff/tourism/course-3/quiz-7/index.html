# 概述、宫殿与坛庙

  1. 曲阜孔庙原为孔子故宅。

    【解释】
    曲阜孔庙。位于山东省曲阜市，是第一座祭祀孔子的庙宇。曲阜孔庙原为孔子故宅。

  2. 中国古代木构架有抬梁、穿斗、井干三种不同的结构方法。穿斗式多用于 等建筑物。

    • 宫殿
    • 寺院
    • 南方民居
    • 坛庙

    【解释】
    “穿斗式” 是用穿枋、柱子相穿通接斗而成，便于施工，最能抗震，但较难建成大型殿阁楼台，所以我国南方民居和较小的殿堂楼阁多采用这种形式。
    【答题技巧】
    “太”（抬梁式）豪华，“穿”（穿斗式）南方。抬梁豪、穿斗小、井干简。

  3. 中国古建筑的发展历史可追溯到

    • 原始社会早期
    • 商周时期
    • 秦汉时期
    • 隋唐时期

    【解释】
    中国古建筑的发展历史可追溯到原始社会早期，原始人利用天然崖洞或构木为巢作为居所。

  4. “左社右祖” 是指宫殿区的左侧是祭祀土地神的社稷坛，右侧是祭祖先的祖庙。

    【解释】
    左祖右社。或称左庙右社。中国礼制思想中的一个重要内容即崇敬祖先、提倡孝道，祭祀土地神和粮食神。所谓 “左祖”，是在宫殿左前方设祖庙，是帝王祭祀祖先的地方，因为是天子的祖庙，故称太庙；所谓 “右社”，是在宫殿右前方设社稷坛。社为土地，稷为粮食。社稷坛是帝王祭祀土地神、粮食神的地方。

  5. 曲阜孔庙是中国礼制建筑规模最大、等级最高的建筑群。

    【解释】
    偷换主体。“曲阜孔庙” 应改为 “北京天坛”。天坛始建于明永乐年间，是明清皇帝祭天和祈祷丰年的地方，是中国礼制建筑中规模最大、等级最高的建筑群。
    【答题技巧】
    孔子（孔庙）也大不过 “天”（天坛）。（孔庙，世界建筑史之奇迹 “唯一的孤例”）。天理（天坛→礼制建筑→最大）。

  6. 历史上许多皇帝，如秦始皇、汉武帝等，都要登 祭祀天地，称为封禅大典。

    • 泰山
    • 黄山
    • 衡山
    • 嵩山

    【解释】
    历史上一些立下丰功伟业的皇帝，如秦始皇、汉武帝等，都曾登五岳之首泰山举行特殊的祭告天地典礼，称为 “封禅大典”。
    【答题技巧】
    五岳之首，泰山。

  7. 吉祥缸是置于宫殿前的防火灾用水缸。

    【解释】
    本题说法正确。吉祥缸置于宫殿前盛满清水以防火灾的水缸，有的是铜铸的。古代称为 “门海”，比喻缸中水似海可以扑灭火灾，故又被誉为吉祥缸。
    【答题技巧】
    吉祥缸是古代的消防拴。

  8. 元大都是按照 传统都城的布局建造的。

    • 汉族
    • 壮族
    • 蒙古族
    • 满族

    【解释】
    元朝的元大都按照汉族传统都城的布局建造，是自唐长安城以来又一个规模巨大、规划完整的都城。

  9. 中国古代宫殿建筑在布局上一般要遵循 原则。

    • 严格的中轴对称
    • 左祖右社
    • 坐北朝南
    • 前朝后寝
    • 主次分明

    【解释】
    宫殿是帝王朝会和居住的地方，以其巍峨壮丽的气势、宏大的规模和严谨整饬的空间格局，给人强烈的精神感染，凸显帝王的权威，是中国古代建筑中最高级、最豪华的一种类型。宫殿的布局一般要遵循：严格的中轴对称、左祖右社、前朝后寝等原则。故本题选 ABD。
    【答题技巧】
    古代建筑特色宫廷是 “前后”（前朝后寝）“左右”（左祖右社）“中”（严格的中轴对称）。

  10. 由一条正脊、四条垂脊和四条戗脊组成，被称为九脊顶的屋顶是悬山顶。

    【答题技巧】
    太 “久” 了（九脊顶），就要 “歇一歇”（歇山顶）。

  14. 中国传统的治国思想是 “以农为本”，发展农业生产与土地密切相关，所以要祭祀土地神和粮食神。

  15. 彩画有防潮、防腐、防蛀功能，下列等级最高的彩画是 。

  4. 中国古代建筑的庭院与组群布局，一般采用对称式，沿着纵轴线和横轴线设计。

    【解释】
    本题说法正确。中国古代建筑的庭院与组群布局一般采用对称的方式，沿着纵轴线与横轴线设计。多数以纵轴线为主、横轴线为辅。一般将主要建筑物布置在纵轴线上，次要建筑物布置在主要建筑物前的两侧，东西对峙，组成一个方形或长方形院落。

    【答题技巧】
    “白送”（北宋）你一套 “法式别墅”（《营造法式》）。

  18. 中国传统梁柱式木构架建筑有 “墙倒屋不塌” 的特点。

    【解释】
    本题说法正确。由于木材建造的梁柱式结构，是一个富有弹性的框架，这就使它还具有一个突出的优点即抗震性能强。它可以把巨大的震动能量消失在弹性很强的节点上。“墙倒屋不塌” 这句民间俗语，充分表达了上述梁柱式结构体系的特点。
    【答题技巧】
    墙倒屋不塌是 “木梁”（梁柱式）撑着呢。

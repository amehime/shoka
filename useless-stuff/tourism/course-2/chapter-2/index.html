# 按技术等级

初级导游
《旅游法》 规定，参加导游资格考试成绩合格，写旅行社立劳动合同或在相关旅游经亚组织注册的人员，可以申请取得导游证，也就是说，具有高中、中专及以上学历，通过国家支化和旅游部组织的统一考试、获得导游资格证书井进行岗前培训，与旅行社订立劳动合同或在相关旅游行业组织注册后，自动成为初级导游。
中级导游
  1. 初级导游报考同语种中级导游和初级外语导游报考中级中 文 (普通话) 导游的，学历不限；
  2. 初级中文 (普通话) 导游和中级中文 (普通话) 导游报考中级外语导游的，必须具备大专及以上学历；
  3. 初级外语导游、中级外语导游报考其他语种中级外语导游 的，必须具备大专及以上学历。
  4. 在取得导游资格证书满 3 年，或具有大专及以上学历的取得导游证满 2 年，且报考前 3 年内实际带团不少于 90 个工作日，经笔试《导游知识专题》《汉语言文学知识》或《外语》，合格者晋升为中级导游。
高级导游
取得中级导游资格满 3 年，具有本科以上学历或旅游类、外语类大专学历，报考前 3 年内以中级导游身份实际带团不少于 90 个工作日，经笔试 《导游能力测试》和《导游综合知识》(包括对旅游政策法规的掌握和运用能力，对旅游业发展趋势的深入了解，对国内外重大事件的及时掌握和分析，以及对旅游相关知识的综合运用能力)，合格者晋升为高级导游。
特级导游
取得高级导游资格 5 年以上，业绩优异，有突出贡献，有高水平的科研成果，在国内外同行和旅行商中有较大影响，经论文答辩通过后晋升为特级导游。

# 按劳动就业方式

旅行社专职导游
在一定时期内被旅行社固定聘用，以导游工作为其主要职业的导游。 这类导游大多数受过中、高等教育，或受过专门训练，为旅行社正式员工，专职为旅行社带团，并由旅行社支付劳动报酬、缴纳社会保险费用。
社会导游（自由执业导游、兼职导游）
社会导游主体是取得导游资格证书，并在相关旅游行业组织（导游协会）注册而取得导游证的导游，但也包括旅行社临时特聘的导游。
社会导游有自由执业导游和兼职导游两类。
1. 自由执业导游。是以导游工作为主要职业，但++并不受雇干固定的旅行社++，而是通过签订临时劳动合同为多家旅行社服务，或者通过导游自由执业平台为散客提供导游服务的人员。自由执业导游是西方大部分国家导游队伍的主体，近年来在我国导游队伍中也占据了主体地位，其主要收入来源是旅行社（或游客）支付的导游服务费。
2. 兼职导游。亦称++业余导游++，是指不以导游工作为主要职业，而是利用业余时间从事导游工作的人员。目前这类导游可细分为两种：一种是通过了国家导游资格考试并取得导游证但只是兼职从事导游工作的人员，他们一般有其他职业，只在空闲时帮助旅游企业带团；另一种是没有取得导游证，但具有特定知识或语种语言能力，受聘于旅行社，临时从事导游工作的人员。兼职导游是我国导游队伍中++一支不可缺少的生力军++。

# 香水使用礼仪

  1. 忌用量过多
    一般情况下， 1 米范围内能够闻到淡淡的幽香较为合适，若在 3 米左右的距离内仍可闻到香味就显得过量了。
  2. 忌使用部位不当
    涂抹香水的部位最好是光线照射不到的地方。如腋下、耳后、手臂内侧等。
    不要搽在手背、额头等暴露部位，比较妥当的办法是在衣领、衣角、手帕上搽一点，任其自然挥发。
  3. 忌不洁使用
    要使香水发挥应有的作用，务必先洗澡，驱除不洁气味。用香水掩盖异味是不正确的。
  4. 忌不同香水混合使用
    不同品牌、不同系列、不同香型的香水不能混合使用，以免掩盖不同香水的香气特点和产生副作用。
  5. 忌吃辛辣剌激的食物
    忌吃葱、蒜、辣椒等。因为食用这类辛辣刺激的食物后容易产生体臭，从而影响香水的使用效果。

# 着装的基本原则

  1. TPO 原则

    T 时间
    男士在白天不能穿小礼服和晚礼服，在夜晚不能穿晨礼服，女士在日落前则不能穿过于裸露的礼服。
    P 地点
    在气候炎热的地方，服装以浅色或冷色调为主，在寒冷的地区，服装则以深色或暖色调为主。
    O 场合
    上班、社交、休闲三大场合。

  2. 配色原则

    黑白灰
    这三种颜色也被称作 “安全色”，上深下浅或上浅下深的原则
    颜色的语言
    如深色的衣服，特别是黑色、深蓝色、深咖啡色等给人以收缩感，瘦人穿着显得更加瘦小，而胖人穿着则会显得苗条反之；浅色的衣服给人以扩张感，适宜瘦人穿着

# 着装的基本要求

与年龄相协调
年龄大些的人喜欢穿深色保守款以显成熟稳重；年龄小些的人喜欢着亮色时尚款以显青春活泼。
与体形相协调
瘦者穿浅色、横条纹、大花图案的衣服可以显得圆润丰满；胖者穿深色、竖条纹的衣服则可显得苗条清秀些；肤色较深的人穿浅色服装会显得时尚健美；肤色较白的人穿深色服装更能显出皮肤的细腻自嫩
与职业相协调
导游是旅游地的形象大使，不宜染头发、穿奇装异服，否则会使游客感觉缺乏亲和力，应该选择适合户外工作特点大方得体的服饰
与环境相协调
在喜庆场合不能穿得太古板，在悲伤场合不能穿得太花哨，在庄重场合不能穿得太随意，在休闲场合不能穿得太隆重。高跟鞋和西服套裙显得高雅大方，适合在参加宴会时穿着，但不适于去登高探险、郊外野营。

# 正装的着装规范

拆商标，烫平整，系纽扣，不卷不挽，慎穿毛衬，少装东西

  • 单排两粒扣式的西服上衣，讲究 “扣上不扣下 “，即只系上边那粒纽扣或全部不系
  • 单排三粒扣式的西服上衣，要么只系中间那粒纽扣，要么系上面那两粒纽扣。
  • 而双排扣的西服上衣必须系上所有纽扣，以示庄重
  • 背心也分为单排扣式和双排扣式两种。
  • 根据着装惯例，单排扣式西服背心的最下面那粒纽扣应当不系，而双排扣式西服背心的纽扣则必须全部系上。
  • 上衣左侧的外胸袋除可以插入一块用以装饰的真丝手帕外，不应再放其他任何东西，尤其不应当放钢笔、挂眼镜。
  • 内侧的胸袋，可以用来放钢笔、钱夹或名片夹，但不要放过大过厚的东西或叮当作响的钥匙串等物。外侧下方的两只口袋，原则上不放任何东西。
  • 在西服背心上，口袋大多只起到装饰作用。除可以放置怀表外，不宜再放别的东西。
  • 在西服裤子上，两只侧面的口袋只能放纸巾、钥匙包或者钱包。其后侧的两只口袋，一般不放任何东西。
  • 衣扣要系上：穿西装的时候，衬衫的所有纽扣都要一一系好。在穿西装而不打领带的时候，必须解开衬衫的领扣。
  • 袖长要适度：穿西装时，衬衫的袖子最好露出西服袖口 2 厘米左右。
  • 下摆要放好：穿长袖衬衫时，不论是否穿外衣，都要将下摆均匀掖进裤腰之内。
  • 大小要合身：除休闲衬衫外，衬衫既不宜过于短小紧身，也不应过分宽松肥大。
  • 鞋袜：与西服配套的鞋子只能是皮鞋，其颜色宜选用深色和单色。
  • 黑色皮鞋可以和任何颜色的西装配套。
  • 男士在穿西服、皮鞋时所搭配的袜子，以深色和单色为宜，并且最好是黑色的。

# 交谈礼仪

  • 基本规则

    1. 委婉含蓄，表达巧妙
    2. 善于倾听。倾听时要集中注意力，要主动反馈，要尊重对方意见
    3. 坦率诚恳，切忌过分客气。欧美人习惯率直地表达自己的意见，日本人交谈时比较含蓄，而且会不时地应和对方
    4. 大方自然。交谈时，要自信、大方、自然，不能扭捏腼腆、惊慌失措或心不在焉
    5. 照顾全局多人交谈时，要照顾大家，要与多数人谈话，不要冷落任何人
    6. 诙谐幽默，避开矛盾的锋芒幽默风趣的话语不仅令人愉快，还能化解由于各种原因引起的紧张情绪和尴尬气氛

  • 忌谈话题

    1. 非特殊场合不要涉及疾病、死亡等不愉快的话题。
    2. 回避对方的隐私。对女士一般不询问其年龄和婚姻情况；对一般人，不直接询问他的履历、工资收入、家庭财产、衣饰价格等私人问题。
    3. 对方不愿意回答的问题不要刨根问底，对方反感的问题一旦提出则应表示歉意或立即转移话题。
    4. 不要批评长辈和身份高的人，不要讥笑讽刺他人，对宗教问题也应持慎重态度。
    5. 不能用词尖酸刻薄，恶语伤人。
    6. 不能用傲慢失礼的话伤害对方的自尊心。
    7. 和外国游客交谈不得胡言乱语或泄露国家机密。

  • 控制音调
    在与人交谈时要注意控制音调，尤其是讲话时尖而响的声音容易引起旁人的反感。
    一个音量适宜、清晰可辨的声音更能吸引人们的注意力并博得信任和尊敬。

  • 礼貌用语
    “谢谢你”、“对不起”、“请” 这些礼貌用语，几乎在任何需要麻烦他人的时候，“请” 都是必须挂在嘴边的礼貌语，道歉时最重要的是有诚意。

  • 交谈的最佳距离和角度
    在社交场合，要注意保持交谈的最佳距离和角度。不同的国家对此有不同的习惯。西欧一些国家认为，两个人交谈的最佳距离为 1 米，但意大利人经常保持 0.3～0.4 米。
    从卫生角度考虑，交谈的最佳距离为 1.3 米，这样就不至于因交谈而感染由飞沫传染的疾病。

    此外，交谈时最好有一定的角度，两个人可在对方的侧面斜站，形成 30° 角为最佳，避免面对面。这个距离和角度，既无疏远之感，又文明卫生

# 举止礼仪

  • 站、坐、走

    1. 导游的站姿要给游客一种谦恭有礼的感觉。是谓古人所说的 “立如松”
    2. 坐于座位的 2/3 位置；
      不满坐是谦恭；
      导游的坐姿要给游客一种温文尔雅的感觉。
      男士膝盖可稍微分开，但不宜超过肩宽；翘腿时，要注意收紧上面的腿，脚尖下压，绝不能以脚尖指向别人；不要抖腿。
    3. 优雅的走：行如风
      导游的走姿要给游客一种轻盈稳健的感觉
      行走时，不要把手插在裤袋里。
      如与人告辞时，不宜扭头便走，示人以后背。为了表示对在场的其他人的敬意，在离去时，可采用后退法。在楼道、走廊等道路狭窄之处需要为他人让行时，应采用侧行步。

  • 鞠躬与蹲姿

    • 鞠躬即弯身行礼，源于中国的商代，是一种古老而文明的对他人表示尊敬的郑重礼节。
    • 另一种是深鞠一躬（15°\～90°），几乎适用于一切社交和商务活动场合，这也是导游最常用的鞠躬方式，为了表达对别人的尊重，都可以行鞠躬礼。
    • 得体的蹲：不走光
      一脚前，一脚后，然后下蹲；
      忌弯腰、翘臀或两脚平蹲（卫生间姿势）
      高低式蹲姿，交叉式蹲姿，单膝点地式

# 人际交往礼仪

# 日常交往礼仪原则

  1. 信守时间

    1. 在有关时间的问题上，不可以含含糊糊、模棱两可
    2. 与游客时间一旦约定，就应千方百计予以遵守，不能随便变动或取消
    3. 对于约定的时间，唯有 “正点 “到场方为得体，早到与晚到，同样都是不正确的做法
    4. 万一失约，务必要向约会对象尽早通报，解释缘由，并向对方致歉，绝不可以对此得过且过或者索性避而不谈，显得若无其事。根据国际礼仪，信守时间，遵守约会，是取信于人的一项基本要求

  2. 不妨碍他人
    在公共场合中，不可以忘乎所以、为所欲为，不可以高谈阔论、大声喧哗。

  3. 女士优先
    女士优先原则的本意是要求每一位成年男子，在社交场合里，都要尽自己的一切可能来尊重、体谅、帮助、照顾、保护，并且随时随地、义不容辞地挺身而出，替女士排忧解难。
    在现实生活中，女士优先的原则是通过一系列的具体做法来贯彻和体现的。例如，在女士面前，男士不应说脏话、开无聊玩笑。

  4. 不得纠正
    是要求我们在同外国友人打交道的过程中，只要对方的所作所为不危及生命安全，不违背伦理道德，不触犯法律，不损害我方的国格人格，在原则上都可以对之悉听尊便，而不必予以干涉与纠正。

  5. 维护个人隐私
    过分关心别人，其实也是对对方的一种伤害，尤其是当发现对方不愿回答时，就应当适可而止。

  6. 以右为尊
    在涉外交往中，一旦涉及位置的排列，原则上都讲究右尊左卑、右高左低。基本上是讲究前尊后卑。

  7. 保护环境
    与西方人交谈时，诸如 “狗肉味道好极了 “一类的话语是很不合适的。

# 日常工作中的礼仪要求

  1. 遵守时间
    按照规定时间提前到达出发地点，如有特殊情况，必须耐心向客人解释以获得谅解

  2. 尊重游客

  3. 注意细节
    外出旅游清点人数时，不宜用手指点；
    车子发动时，要提示客人坐稳；
    行车时一般可致欢迎词；
    客人到酒店时，提醒客人携带好随身物品；
    导游讲话时音调柔和清晰，音量适中，手势简练，举止大方；
    在参观游览时，要注意游客的兴趣选择性讲解；
    欢送客人时，要致欢送词；
    游客邀请品尝风味小吃时，切忌主宾颠倒，要注重进餐时的礼仪等

# 人际沟通的礼仪

称呼礼仪
指的是人们在人际沟通中所采用的称谓语

  1. 生活中的称呼

    • 敬称：对于有身份者、年长者，可以称呼其 “先生”。其前还可 以冠以姓氏，如 “李先生”。对文艺界、教育界人士，均可称之为 “老师”。在其前，也可加上姓氏，如 “王老师”。对德高望重的年长者，可称之为 “老”。将姓氏冠以 “老” 之前，如 “李老”
    • 姓名的称呼：平辈的朋友、熟人，彼此之间均可用姓名相称。为了表示亲切，可免称其名。对年长者，称 “老刘”；对年幼者，可称 “小王”

  2. 工作中的称呼

    • 导游在工作中称呼游客应注意庄重、正式、规范。
    • 称呼职务：是一种最常见的称呼方法。具体分为 3 种情况:
      1. 仅称呼职务，如 “经理”“主任”“院长” 等。
      2. 在职务之前加上姓氏，如 “赵处长”“李书记” 等。
      3. 在职务之前加上姓多，这仅适用极其正式的场会，如 “王鑫 总经理” 等
    • 称呼职称
      称呼具有专业技术职称者，尤其是具有高级、中级职称者，可以在工作中直接以其职称相称，如 “律师”“医生”“教授” 等 职称前加上姓氏，如 “欧阳编辑”“王调研员”。也可以简化称呼，如将 “孙工程师” 简称为 “孙工”。还可以在职称前加上姓名，这种适用于十分正式的场合，如 “张 X 教授”“汪 X 主编” 等
    • 称呼学衔
      称呼游客的学衔，可增加被称呼者的权威性。可以仅称呼学衔，如 “博士”。也可在学衔前加上姓氏，如 “蒋博士”。可以在学衔前加上姓名，如 “关 X 博士”。还可以将学衔具体化，说明其所属学科再加上姓名，如 “物理学博士王 X”“人类学博士梁 XX” 等，这种称呼最正式
    • 称呼行业
      直接称呼职业，如称呼教员为 “老师”、称呼教练员为 “教练”、称呼警察为 “警察” 等
    • 称呼姓名
      称呼姓名仅限于同事、熟人之间。可以直呼姓名，也可以只 呼其姓，不称其名，但要在其前面加上 “老”“大”“小”， 如 “老马”“小黄”。还可以只称其名，不呼其姓，这种称呼通常限于同性之间，尤其是上司称呼下级、长辈称呼晚辈时

  • 握手礼仪

    • 握手顺序

      • 男女之间，男方要等女方先伸手
      • 女方不伸手且无握手之意，男士可点头或鞠躬致意
      • 宾主之间，主人应先向客人伸手，以表示欢迎
      • 长辈与晚辈之间，晚辈要等长辈先伸手
      • 上下级之间，下级要等上级先伸手以示尊重

    • 时间：初次 3 秒

    • 力度：在一般情况下，握手不必用力，握下即一可。男士与女士握手不能握得太紧，西方人往往只握一下女士的手指部分，但老朋友可例外

    • 禁忌

      1. 忌多人同时握手，忌交叉同时与两个人握手；
      2. 忌精力不集中，握手时看着第三者或者环视四周；
      3. 男士握手忌戴手套；
      4. 忌将左手放在裤袋里；
      5. 忌紧握对方的手，摇来摇去，长时间不放；
      6. 忌边握手边拍对方肩头，握手时低头哈腰或与他人打招呼。

  • 介绍礼仪

    • 自我介绍：时机内容分寸，半分钟左右为佳
    • 他人介绍：顺序内容应对。尊重优先，少数服从多数。
      • 应遵守 “尊者优先了解情况” 的规则，大致有以下几种情况
        • 进行集体介绍时，首先强调地位、身份，当双方地位、身份大致相似的时候，则遵循 " 少数服从多数的原则，即先介绍人数少的一方，后介绍人数较多的一方
      • 介绍顺序
        1. 先介绍年幼者，后介绍年长者
        2. 先介绍晚辈，后介绍长辈
        3. 先介绍男士，后介绍女士
        4. 先介绍家人，后介绍同事、朋友
        5. 先介绍主人，后介绍来宾
        6. 先介绍职位、身份低者，后介绍职位、身份高者
自我介绍
应酬式一般场合，旅行途中或宴会中。你好，我叫 x
工作式适用于工作中。1. 姓名 2. 所在单位 3. 职务。你好，我叫 xx，是 xx 公司人事经理
交流式适用于社交活动中，希望对方认 识自己了解自己，建立联系。我叫 xx，和您先生是大学同学
礼仪式适用于讲座、报告、演出、庆典、仪式等隆重场合。 可加入一些适宜的谦辞、敬语。各位来宾，大家好！我叫 xx，是远程公司的总经理。 我代表本公司热烈欢迎大家光临我 们的产品发布仪式，谢谢大家的支 持
问答式用于应试、应聘和公务交往。您好！不知道应该怎么称呼？经理好，我叫 xx
为他人介绍
标准式以姓名、单位、职务等为主
简介式内容只有双方姓名甚至只有姓
强调式介绍姓名，还强调特殊关系，引起重视
引见式普通社交场合，介绍者把被介绍双方引导在 一起，无实质内容
礼仪式正规标准，但语气表达称呼更礼貌、谦虚
推荐式使劲夸优点。
  • 电话礼仪
    1. 时间适宜
      白 8 点后、 9 点后，晚 22 点前注意时差， 3 分钟原则
    2. 内容简洁 5W+How
    3. 体现文明
      响 3 声接，等对方挂机，不随意接打电话，会议静音

# 第三节 导游的修养、行为规范及职业道德规范

旅游行业精神文明建设的目标是：以 “全心全意为旅游者服务” 为宗旨，立足岗位做奉献，全面提高从业人员的政治和业务素质，逐步规范行业行为，树立起积极、健康、文明的旅游行业新风尚。

# 导游的修养

  1. 知识修养
    • 学风修养和文化修养
      1. 勤学、善学、不耻下问
      2. 丰富，知识、艺术鉴赏能力、兴趣爰好、审姜情趣和礼节礼貌
    • 广博的知识是导游立业之基。导游人员的知识修养包括学风修养和文化修养。
  2. 情操修养
    • 执着的信念和追求
      1. 对国家
      2. 对集体
      3. 对游客
      4. 对工作
  3. 气质修养
    • 气度、脾气、秉性或性情
    • 一方面，严格按照计划，不允许有明显的个性表现
    • 另一方面，体现彬彬有礼、落落大方，对客服务要主动、热情、勤勉、灵活。

# 导游的行为规范

  1. 严守国家和企业的机密，注意内外有别
    导游人员不得有损害国家利益和民族尊严的言行，不得擅自带领游客进入保密禁区、军事要地参观、游览；不得泄露旅游团收费细目，在游客面前，不谈论内部情况，在涉外场合，不携带内部文件。

  2. 严格遵守请示报告制度
    遇到重大情况和问题（如治安事故、交通事故等），要及时汇报，非紧急情况不得擅自决定或处理

  3. 遵纪守法
    10 人以上团队应打接待社社旗

  4. 自尊自爱，不失人格、国格

    1. 不得 “游而不导”，不擅离职守，不懒散松懈，不本位主义，不推诿责任。
    2. 关心游客，不态度冷漠，不敷衍了事，不在紧要关头临阵脱逃。
    3. 不与游客过分亲近；不介入游客内部的矛盾和纠纷，不在游客之间搬弄是非；对待游客要一视同仁，不厚此薄彼。
    4. 有权拒绝游客提出的侮辱人格尊严或者违反其职业道德的不合理要求。
    5. 不得迎合个别游客的低级趣味，在讲解、介绍中掺杂庸俗、下流内容。

  5. 注意一些小节

    1. 不随便单独去游客的房间，不单独去异性游客的房间。
    2. 不得携带自己的亲友随旅游团活动。
    3. 不与同性外国旅游团领队同住一室。
    4. 饮酒量不超过自己酒量的 1／3。
    5. 不克扣客人餐费。
    6. 不私自留用旅行社送给客人的礼品。

# 导游的职业道德

# 道德与职业道德

  • 道德是一种社会意识形态，是在一定社会中调整人与人之间以及个人与社会之间关系的行为规范的总和。

  • 职业道德是指从事某种职业的人员在工作或劳动过程中应遵守的与其职业活动紧密联系的道德规范和原则的总和。

# 导游职业道德

  • 导游职业道德是导游在工作过程中应遵守的与其职业相适应的道德原则和道德规范总和。——— 权利和义务

  • 1996 年 11 月，国家旅游局制定了《加强旅游行业精神文明建设的意见》社会主义旅游事业进入一个新的历史时期。

  • 主要内容：

    1. 爱国爱企、自尊自强
    2. 遵纪守法、敬业爱岗
    3. 公私分明、诚实善良
    4. 克勤克俭、宾客至上
    5. 热情大度、清洁端庄
    6. 一视同仁、不卑不亢
    7. 耐心细致、文明礼貌
    8. 团结服从、顾全大局
    9. 优质服务、好学向上

  • 主要内容解析：

    • 爱国爱企、自尊自强不仅是导游必须遵守的一项基本道德规范，也是社会主义各行各业必须遵守的基本行为准则。
    • 克勤克俭、宾客至上是导游处理与游客关系的一条基本行为准则。
    • 热情大度、清洁端庄是导游在接待游客的过程中应当具备的基本道德品质和道德情操。
    • 耐心细致、文明礼貌是导游一项最重要的业务要求，它是衡量导游工作态度的一项重要标准。
    • 团结服从、顾全大局是集体主义原则在导游工作中的具体体现
    • 衡量导游道德素质高低的标准是看其是否具有优质服务的意识

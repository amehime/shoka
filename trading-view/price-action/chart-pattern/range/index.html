# 交易区间的形成和特点



  • 市场在进入交易区间之前，大多现有急速的趋势性运动

  • 趋势运动，多空主力观点一致，一方持续入场，一方暂时退出观望。

  • 区间运动，多空主力出现分歧，顺势方想把价格推动到趋势极点，逆势方认为当前价格值得参与，持续入场。

  • 换个角度理解：交易区间是多空双方都认为有交易价值的区域，而趋势运动就是市场在寻找这个价格区域



  • 如果价格突破一个价值区域后，运行了很远才形成新的价值区域，这是大牛市或大熊市

  • 如果价格突破一个价值区域后，没运行太远就停顿，形成了新的价值区域，再突破回到原来的价值区域内，这就是市场从一个小区间变成了一个大的交易区间。多空主力对市场运动方向没有形成一致的观点，双方仍然在大的价值区域内积极参与

  • 上行通道或者下行通道都是倾斜的交易区间，多空主力在通道中都在积极参与市场，其中某一方占有略微的优势，但不是绝对优势

# 如何在图表上判断交易区间的形成


  • 图 1 中，先有一次多头的强势推动，这最后的推动部分是一个通道

  • 在多头趋势推动到极值之后，市场进行了深幅的回调，其实这次回调就是一次急速与通道，然后对通道起点的回测

  • 在深幅回调之后，多头趋势延续，但是这次延续并没有对趋势方向进行明确的突破，而是趋势方向的突破失败了，价格再次进行调整

  • 这次调整在前面的波段点处获得了支撑，当价格在前面波段点处再次获得支撑时，则认为市场形成了一个潜在的交易区间

    • 如果价格没有获得支撑，而是强势跌破了，那它就是一个潜在反转形态

  • 总结形成区间的简单步骤：

    • 趋势的回调（最好是深幅回调），顺势无法成功突破（趋势力量减弱）
    • 下一次回调在前波段点处获得支撑（如果此时跌破前波段点，就是潜在反转形态）这样就可以认为市场将进入一个潜在的交易区间。

# 交易区间的特点

  • 价格运动具有惯性，一定要牢记这一点。

  • 无论是趋势运动还是区间运动，市场都倾向于维持当前运动状态。

  • 所以在明确的趋势中，潜在反转形态大多都失败，演变成趋势延续形态；而在明确的区间中，大多数对区间的突破都会失败，变成假突破。

# 如何交易区间

简单来说区间的交易就分为两种：

  • 一种是区间内交易，就是常说的高空低多，或者说区间假突破。这些交易的前提是认为区间会持续运行下去
  • 另一种就是交易区间的突破。不再认为区间运行下去了，认为已经从区间转为了趋势

# 区间的交易逻辑

# 区间内交易

只要一个明显的区间形成，高空低多

只要区间状态持续，就能一直盈利

市场运动状态具有惯性，倾向于维持当前的状态，所以一旦区间形成，假突破的概率是大于真突破的，所以做区间内交易就具有优势，即便不需要信号 K 线的验证

不经过 K 线验证属于左侧交易，并不是很安全

# 形态周期选择

在组成 K 线数少的形态结构中，可以找信号 K 线入场交易：单 K 线反转，双 K 线反转，二次突破

在组成 K 线数多的形态结构中，比如几百根 K，这种情况最好找入场形态：双重顶底，头肩顶底，楔形顶底，箱体突破

大周期的单 K 线反转，双 K 线反转，二次突破就是小周期的双重顶底，头肩顶底，楔形顶底，箱体突破



看喜欢用哪个方式，如果喜欢信号 K 就选择 K 线数少的，喜欢形态就选择 K 线数多的
但不要在更大周期找信号 K 线入场

# 区间突破失败

在区间突破时，价格出现停顿，它是代表了区间突破成功还是失败？我们事前永远无法得知，无论用什么指标什么判断方式都是一样的。

所以做区间突破失败的交易，就看它在突破时是否出现乏力，是否没有跟进力量，只要出现衰弱的信号，就可以主观认为它突破失败了

如果市场先出现突破乏力的信号，再继续突破，那突破失败的信号就失败了，也就是区间突破失败的失败，也就是突破回调

在盘中我们只能根据出现的信号作出决策，一旦市场否定了我们的判断，那就严格止损，不要和市场较劲

# 潜在交易区间形成

  • 一段急速的下跌趋势形成后，很难被改变
  • 但会有深幅回调出现，回调后，市场会对趋势极点进行测试
  • 如果市场出现了犹豫，没有按预计下跌测试极值，则形成了交易区间，代表了不确定性，不确定则容易带来假突破

# 底部入场时机

  • 如果距离下一个支撑或下一个重要的点位很近的时候，可能会先否认前面的入场机会，去测试这个很近的下一个的支撑或阻力
  • 交易区间中，高空低多是有优势的
  • 当价格到达区间中间，就是一个中性的概率状态，可以平掉一半保住利润，剩下一半移动止损到开仓位置
  • 区间中部是个非常混沌的状态，不应该开单

# 顶部入场时机

  • 当价格运行到了区间的顶底部，给了入场做空的形态，形成了两个低点
    • 但最好是等对区间顶部有假突破后再入场
  • 后续形成了铁丝网形态，代表了极端的交易区间，代表了市场的不确定性，当入场后出现了这样的形态，最好原位止损或微亏离场
  • 也可以在该窄幅区间的上方挂单入场，窄幅区间的上方也是大区间的上方，做空仍有数学优势

  • 市场后续仍然将上述做空点位全部否定，打到止损；
  • 出现了一根长影线的信号 K 线，向上假突破了大区间和窄幅区间，突破是无力的，没有跟进力量；
    • 该 K 线影线太长，止损位很远，不建议直接入场，应该在 K 线的中上部挂单
  • 后续依然是窄幅区间，形成了复杂的运动状态
  • 尝试做区间突破失败，主观上认为到区间顶部应该做空，因此继续寻找做空的入场点
  • 区间中部不建议做单，区间内的大阳线大阴线都是没有意义的

# 再次到达底部

  • 依旧寻找做多的逻辑
  • 有假突破最好
  • 出现高 1 后，高 2 迅速向上突破，可能来不及入场，需要快速读图的能力
  • 区间中部止盈一半，并将移动止损设到开仓位置
  • 因为不是每次都会到区间顶部再反转

# 区间内部潜在交易位置

  • 区间内部形成新的水平波段点，仍然可以看情况交易
  • 区间最终会被突破，应该注意到中部时止盈，以及移动止损
  • 要站在有概率优势的一方

# 总结

注意点：

  1. 市场首先要是一个潜在的交易区间；交易区间没有形成，自然没有突破和突破失败一说
  2. 最希望的一种市场状态是，希望它出现急速的向区间边界的运动，然后对区间边界造成假突破
  3. 寻找信号 K 线，比如二次突破或者强的单 k 线反转

# 区间突破

  • 价格在突破一个波段点，或者突破区间多次验证的水平位的时候，或者突破一个倾斜的支撑阻力线的时候，一般会有几种运动方式
  • 应该认清这些突破方式的区别，有些突破是不应该去参与的，比如区间的急速突破


  • 区间的急速突破，从起涨到结束都是连续的急速的，价格没有过多的停顿

  • 这种急速的运动，它可能是一种高潮运动，或者是一种真空效应，一般会带来假突破

  • 而且我们很难设置一个合理的止损位，总之尽量不要交易这种突破方式

    • 止损点一般是技术支撑位，即上一个波段点，或者价格密集区的底部
    • 就算放在突破 K 线的下方都很远

  • 推荐的突破入场模式：

    • 突破 - 回调 - 突破
    • 突破前压力

# 突破 - 回调 - 突破

  • 趋势线的突破具有多解性，不怎么好操作
  • 而使用突破 - 回调 - 突破的模式，可以有比较明确的止损位和入场位

# 通道

  • 价格在突破后又走了三根 K 线
    • 第一根是比较强的阳线，观望
    • 第二根是一根代表空头力量的 K 线（有长上影线，有比较长的空头实体），发生在前波段高点位置附近，关注
    • 一次急速的向上运动，在前波段高点位置停顿出现阴线，代表多头动能衰竭，可以视作做空信号，突破可能失败，即形成假突破
    • 但第三根 K 线是阳线，可以看做突破失败的回调 / 失败，并形成一个新的波段低点，可以当做止损点
    • 如果主观认为突破成功，则多单入场放在二号和前波段点构成的水平线上，等待突破成功后入场

# 收敛三角


  • 收敛三角是一种幅度逐渐缩窄的交易区间，上下边界都被固定了，交易此区间突破会相对容易
  • 更多关注前波段点的上下边界，尽量少交易趋势线的突破
  • 如果下破了，入场止损位在前波段点的上方
  • 如果上破了，入场止损位在前波段点的下方
  • 大周期上看似乎没有突破 - 回调 - 突破的模式，直接一根大阴线进行了突破
    • 此时这个模式可能发生在小周期上，可以调到 1/4 周期查看
    • 先有一个向下突破的阴线，其极值可以设为入场位；若后续再突破该极值，则收敛三角下破成功
    • 随后跟随了一个回调的阳线
    • 第三根是一个强势阴线下破，阴线的高点可以设为止损位

  • 交易区间末端出现收敛形态，对收敛形态的突破就是对交易区间的突破
  • 入场位在突破 K 线的下方
  • 止损位在收敛三角的上缘

# 突破前压力

  • 价格从区间底部急速上涨到区间顶部，急速上涨带来动能衰竭，在区间顶部形成了铁丝网形态
    • 此时做多或做空可能都有理由，看主观方向是哪里
  • 如果是主观突破做多的交易者，当看到一个价格的密集区，假设价格再次上破的话，此处就是一个很好的防守位，只需把止损放在价格密集区的下面
  • 如果急速上涨后，没有出现密集区，则缺少了这样明确的止损参考点
  • 可以把区间顶部或底部的铁丝网形态 / 价格密集区，理解为突破前的压力
  • 如果主观突破失败，这也可以理解为突破失败的压力蓄积，止损可以放在价格密集区的上面
  • 如果突破成功，在该密集区交易的空头都会止损出场，他们的出场会进一步推动价格上涨
  • 该形态最好是一个有明确的上边线和下边线的小箱体，交易突破成功和突破失败都可以利用它来设定止损位置

  • 价格急速上涨，不建议去交易，因为很容易被套
  • 但如果急速上涨后出现横向调整，小箱体，微型交易区间
  • 微型箱体的突破，就可以顺便交易大区间的突破，也可以将其视为突破前的压力
  • 再突破就有了突破入场做多的位置，和明确的止损位

当价格运动出现停顿，是突破成功还是突破失败，在市场走出来之前完全是主观的
第一，需要入场逻辑
第二，需要严格止损
第三，交易的优势从来都不是来源于胜率，而是盈亏比

# 突破后的震荡

  • 突破之后的市场也可能出现犹豫
  • 一段明确的交易区间，最后价格急速突破了区间，并且已经大幅突破了前面的波段低点，这个幅度可以认为是突破成功的
  • 但是突破成功后并没有马上延续下跌，而是又做了一个小箱体
    • 这种小箱体在更小时间周期，就是一个小交易区间
    • 在这个小箱体被明确上破下破之前，突破成功和失败的概率各占 50%
    • 如果主观认为是突破成功，那么小箱体可以认为是突破后的调整
    • 如果主观认识是突破失败，那么小箱体就可以是突破没有跟进力量的表现
    • 到底是什么，只有这个小箱体被突破之后才知道
  • 在价格继续下破区间时，急速运动不做单
  • 下破之后的箱体，可以提供一个明确的止损位和入场位
    • 在箱体下破时入场做空
    • 止损位就在箱体之上
    • 甚至可以去小箱体的更小时间级别去找突破 - 回调 - 突破，止损可能更窄

# 理论知识

一轮明确的趋势在什么情况下才会反转？

只有当足够多的顺势力量平仓离场或不再入场（趋势能量不足），加上足够强的逆势力量出现，才可能迎来一轮大趋势的终结反转。

如果没有强大的逆势力量入场，或只有逆势力量入场，而顺势方没有放弃抵抗，市场可能不会迎来反转，而是进入一个震荡区间，多空主力重新去博弈，决定后市走向。

不少的趋势反转，都源自于顺势突破失败。

因为当价格做了回调整理形态后，再向趋势方向发生突破并且失败，这是趋势能量不足的表现。

一个交易员所能掌握的最重要的技能之一，就是能够准确判断一次突破行情是成功引出一波新的趋势，还是最终突破失败使得行情反转。

这个突破不仅限于水平位，也可能是通道线或者其他支撑阻力结构。

各种时间周期中的趋势反转，都发生在支撑位和阻力位上。

一个成功的趋势反转，能将多头市场转变为空头市场，或将空头市场转变为多头市场，但是必须要记住最重要的一点是大部分趋势反转的尝试都将无功而返。

我们常讲市场运动具有惯性，倾向于维持当前的运动状态，不会轻易改变。

其实根本不存在所谓的趋势反转形态，当市场处于一波趋势行情时，所有的形态都是持续整理形态，也就是中继形态。

但偶尔某个持续形态会失败，表示出顺势力量枯竭。

大多数技术分析师都会把失败的持续形态，定义为反转形态，但大部分情况下这些形态都不会引起价格反转。

例如，在实际交易中，大多数的双重顶底、头肩顶底形态带来的不是反转，而是交易区间。不代表出现这些形态市场就倾向于反转。

大部分反转形态都是震荡区间，但由于 8 成概率反转形态都会失败，所以区间顺势突破的概率会大一些。

在做反转交易时有一个非常重要的原则：
绝对不能在重要的趋势线被突破之前就考虑逆势交易。
此外，只有在出现强劲的信号 K 线时才能做反转交易。
需要证据表明逆势的一方足够强大到去把握掌控市场的机会。

即便如此，仍然应该寻找顺势交易的机会。
因为在第一波逆势浪潮之后，市场大概率会测试原趋势的极值，尝试回归原有的趋势。
只有在极少数情况下，趋势线的突破有非常强劲的动能，对原趋势极值的测试幅度很小就延续了逆势波动。

因为只有价格两次在前趋势极值附近失败了，市场才会做出相反的尝试。

也就是说：突破主要趋势线是一个条件，对前趋势极值测试失败是一个条件，最后一个条件就是一次逆势的突破。

前两个条件证明了多头力量不足，而最后一个条件才代表了逆势方主导市场。

一般认为反转是从牛市到熊市或者从熊市到牛市，但震荡区间的行为也可以算是趋势行为的对立面。

如果一段震荡区间突破进入趋势行情，或者趋势行情转变为震荡区间，也可以算是市场行为的一种反转。

趋势的回调可以是小型的震荡区间，或者是一波与大趋势相反的小规模趋势，也就是常说的回调。

当回调结束时，小规模趋势便会反转，回归到原来的主趋势。

大部分的趋势反转都是更大时间周期上的趋势回调。

大部分趋势反转尝试的结果并非是强劲的反向趋势，而是区间震荡，交易者绝不可能事先知道当前的反转是否会引出一轮新的逆势趋势行情。

因为进入一个区间震荡的反转，往往和进入一轮新趋势的反转，在前十几根 k 线的时间里看起来是一样的。

只有当价格向下突破了楔形起点的波段水平位，才知道这个反转形态成立了。

假如市场在这个水平位获得了支撑，做一个横向的区间震荡，也是完全有可能的。

在价格向下出现逆势突破之前，我们不知道这次逆势突破到底是否会成功，它到底是带来一轮新的逆势趋势，还是做一个横向的区间。

只有在市场运行很长时间后，才能看出来。

但随着走势越来越明确，交易的成功率越来越高，但是剩余的盈利空间也越来越小。

# 反转形态

  • 注意事项：
    1. 市场运动具有惯性，倾向于维持当前的运动状态，在一轮明确的趋势中，无论多完美的反转形态，胜率都很低，大多数情况是形成区间，而不是反转趋势。
    2. 大多数情况下，建议只在趋势回调末期，根据反转形态入场，参与到大的趋势方向中，或者在重要的支撑阻力位，把反转形态当作支撑阻力的验证。
    3. 反转形态的逻辑非常简单，但在实际交易中去应用掌握会很难，做反转很多时候都需要依赖经验，进行主观判断。

# 主要趋势反转形态：双重顶底、头肩顶底

  • 主要趋势反转的步骤：
    1. 当前的行情必须是一轮趋势性的走势
    2. 一次强势的逆势运动，突破重要趋势线和均线
    3. 对原趋势极值进行测试，然后开始第二次逆势运动，这次测试的极值可以更高更低或平行
    4. 第二次的逆势运动必须走得足够深远，带来逆势的突破，波动幅度使市场达成一个趋势已经反转的共识

复盘中看，最佳入场位置是头部和右肩，但是在实操中并不容易

此例中，实盘头部因为各种不确定性，并没有好的入场机会。

右肩是头部的二次入场机会，如果市场给二次入场机会，那么会形成头肩顶，如果不给二次入场机会，就是双重顶。

此时仍然没有发生逆势突破，那我们就还不能认为趋势它已经反转了

一般按交易区间逻辑做单时，止损放在信号 K 线另一端，但如果主观认为是走潜在反转形态，那么止损应该放在头位影线的上方，才是一个有效的单子，但这样盈亏比会下降很多；而缩窄止损，又会导致胜率进一步降低。

如果下跌到前波段低点（颈线），就平掉半仓，剩下的半仓再拿着去等一个逆势突破的出现。

因为逆势突破出现之前，不能认为这个反转形态已经成立。
如果反转不发生，走了震荡区间，有可能会利润回撤，故应先保住一部分利润。

第三个交易点是逆势的突破位，去交易波段支撑水平位（颈线）的突破，具体的入场形态和止损位的设置和区间突破是一样。

必须要有第三个条件出现，反转才算成立，否则可能多头夺回主导权，继续原趋势发展，反转失败。

一般情况下，大约有 40％的概率可以吃到一个有较大盈利空间的逆势波段，有 30％的概率带来小幅亏损，30％的概率产生小幅盈利。大部分入场的结果都是震荡区间而非趋势反转。


获利目标位：测算价格运动目标一般都取这个形态的高度，突破后做等距延长

# 三推（楔形）反转



同样也需要上述的三个反转条件

容易交易的三推形态的共同特征：有一段强势的推动，然后带两段明显变弱的推动，这样的形态可以明显感觉到趋势力量变弱。

假设三段推动都是一样强劲的运动，那就可能形成一个宽幅的趋势通道，此时最好不要以三推反转的形态来对待它。

通道分为紧凑型通道，窄幅通道和宽幅通道，代表不同的市场力量
所有的上行通道都倾向于下破，即熊旗，但是下破之后能走多远，是否值得交易，要具体情况具体分析

三推从哪里开始数有多解性，一般来说第一波推动之后，第二步的推动也非常的强势，那就把第二波推动当做起点开始数。

如果这波牛市趋势非常的强劲，那么做多交易者就会在突破一个高点时再次买入。
如果市场上的交易者发现突破时发生了抛售，他们就会知道做多交易者并没有在突破时继续买入，而是卖出获利平仓。
给出的信号就是，强劲的做多交易者也对市场会持续上涨失去了信心。
当趋势能量明显减弱时，市场就会面临深幅回调或者进入震荡区间，甚至发生趋势反转。

三推反转的入场方式和楔形回调一样：

  1. 第一种是在通道线挂单入场，价格碰到通道线，或者等价格向上假突破时挂单入场。
    上涨趋势中，下面连接波段低点组成的线是趋势线，上面连接波段高点组成的线是通道线。
    通道线也可以通过将趋势线平移来绘制，也就是价格通道。
    此方法有很好的价格优势，盈亏比高，胜率也不会很低。
    但很多时候价格并不会碰到通道线，很少机会能挂单入场。
    如果有对水平支撑线的假突破也可以入场。
    止损的设置可以使用 ATR 辅助。

  2. 第二种是在突破趋势线的时候入场。
    不太推荐，因为大多数的趋势反转都会对原趋势极值进行一个测试，如果在突破趋势线的时候直接入场，那后面再对趋势极值进行测试的过程中，持仓可能会浮亏。
    如果是宽幅通道，价格到达趋势线时，已经运动了非常大的幅度了，再入场获利有限。
    如果用突破回调再突破的模式，缩窄止损去交易这个突破，做个短线持仓，获利平仓后，等价格向上的回撤，然后再继续去做空，则勉强能接受。
    只有在楔形已经走得非常窄的时候，可以考虑在突破趋势线的时候直接入场，因为此时入场止损放在通道线上方，非常窄。
    此外，如果有对水平支撑阻力位的突破更好，因为水平支撑阻力的有效性远高于倾斜的趋势线。

  3. 第三种是突破回测时入场。
    场内的主要参与者需要在这次回测走势中看到趋势方向的主力疲弱了，逆势方才会大举入场。
    回测入场就是常说的二次入场机会，但不一定每次都有。
    难点在于不知道价格到底会回测到哪里，可以使用斐波那契回撤工具。
    或在关键的价格水平位出现强势 K 线时才会考虑入场。

三推形态的获利目标：

  1. 第一目标是楔形的起点，也就是第二次推动的起点，平仓一半或全平
    这个位置也是趋势方可能重新入场的一个位置，市场可能会进入震荡区间

  2. 第二目标是急速的起点，也就是第一次推动的起点，全平

所有反转形态的成功率都不会很高，三推反转也有很大失败的可能

大多数的交易形态都可以分成这三种入场方式：

  1. 挂单入场（左侧入场）
  2. 突破入场
  3. 回测入场（右侧入场）

# 高潮反转

高潮反转和末端旗形这两种反转模式很难入场交易，更多是用来管理已经在持仓的订单。

高潮反转是市场极端情绪的表现。

市场可能已经进入超买状态，超买状态就是多头趋势运行很久，没有明显回调。


有两条通道线，关注通道线的斜率，第二条通道线的斜率明显增加，这是市场在加速的表现，而且这种斜率增加的幅度，说明市场的买入情绪已经非常的迫切了。

市场极度贪婪，是指场内参与者担心市场不会再给到好的回调价格，愿意以任何市价买入，它的表现就是连续的强趋势 k 线，而且趋势 k 线越来越长，并且对短期的趋势通道线进行突破。

然后价格急速反转，大幅下跌，而当它跌到一个多头愿意重新入场的价格水平之后，那会再次去测试前高。

测试前高无力，又可能做一个新的整理区间，也是一个收敛形态。

当这个新的形态再次下破之后，整个的趋势才彻底的反转。

# 末端旗形

不建议新手使用。


末端旗型的定义是引出趋势最后一段推动的调整形态，阿尔布鲁克斯管所有的整理形态都叫旗形，不一定是非要是通道。

图中，红色箭头标注的都是趋势整理形态，绿色箭头标注的都是趋势方向的波段运动。

这些调整形态引出的趋势波动都是很弱的，带来的都是假突破，甚至都没测试到前趋势极值。

例 1 一个旗形调整，前面的一段大的下跌调整之后一段小的下跌，小的下跌是无力的，然后再一个末端旗形反转。

# 总结

我们无法准确预知形态的变化，做交易的核心在于对策

  1. 入场后止盈止损的设置（取决于自己的交易系统）
  2. 入场后直接打到止盈止损，这种情况不需要订单管理
  3. 入场后市场横盘
  4. 入场后价格向止盈目标运动没多远就停止并且反向
  5. 入场后价格打了止损但马上给了很好的二次入场机会

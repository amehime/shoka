# 概念

鲨鱼模式是一种新的谐波模式，本质上，模式是在 5-0 形成之前的主要结构。
该结构在典型的 M 和 W 框架之外。
它具有一种称为 “极端和谐脉冲波” 的独特形式，可以重新测试定义的支撑 / 阻力，同时收敛于 0.886 回调线–1.13 扩展区域。
在所有情况下，完成点都必须包含强大的 0.886 支持 / 回撤作为最低要求。
此外，独特的极限谐波脉冲波采用最小 1.618 扩展。
这与 0.886 结合的回撤线定义了一个独特的结构，该结构具有两个深刻的和谐度量来定义最低水平。

在许多情况下，价格行动将重新测试该模式的初始起点，并定义极好的机会，以利用在有限的时间内移动速度过快的市场。
这些结构通常具有过度延伸的价格行为，该价格行为利用了极端谐波脉冲波的 “自然” 状态，该状态通常按反应性交易行为进行分类。

这种模式通常定义了极好的机会，但是这些逆转通常非常剧烈，并且需要特定的管理策略来利用这种现象。
在许多情况下，价格走势将重新测试先前的支撑 / 阻力位，通常会导致有限的逆势波动。
这种结构的处理方式应不同于标准的 M 和 W 模式。

实际上，对价格行动的总体期望应该是短暂的，并试图抓住最明显的机会。
鲨鱼模式会产生许多准确而积极的反应，只要应用更加积极的管理，这些反应就可以成功进行交易。

# 形态要素

  • 跟其他 M／W 形态比较，这个形态区别在 C 点超越 A 点，反转区也有不止一个区域
  • 非常重要的是 BC 延伸必须打到最少 1.618，没有达到这个延伸标准的形态不是合格的形态
  • 反转区可能是 XC 0.886，XC 1.0 或 XC 1.13，由 2 条斐波那契线组成

# 形态确认

操作步骤：

  1. 【斐波那契回撤】工具，从 X 到 A，寻找 B 点
    • B 点一般在 XA 0.382 - 0.618 之间
  2. 【斐波那契回撤】工具，从 A 到 B，寻找 C 点
    • C 点必须在 AB 1.13 - 1.618 之间
    • 如果在 1.13 附近反转，则是经典的假突破
  3. D 点反转区（PRZ）只需找 BC 延伸和 XC 回调的重叠区。
    反转区由这两条斐波那契组成：
    1. XC 0.88 - 1.13 (XC 0.886，XC 1.0，XC 1.13)
      • 【斐波那契回撤】工具，从 X 到 C，分别在 XC 0.886，XC 1.0，XC 1.13 画水平线
    2. BC 1.618 - 2.24
      • BC 1.618 如果已经超越上述 XC 0.886，则舍弃 XC 0.886
      • 剩余的水平线分别与最接近的 BC 斐波那契线构成反转区

很多时候 BC 延伸可能超过 2.24，到达 2.618 或 3.14。
BC 延伸不太重要，比较关键的是跟 XC 0.886 - XC 1.13 是否有重叠，形成反转区。




# 止盈止损

  1. 止损：

    • 【斐波那契回撤】工具，从 X 拉到 C
    • 如果 0.886 XC 进场，则止损放在 1.0 XC + 少许空间
    • 如果 1.0 XC 进场，则止损放在 1.13 XC + 少许空间
    • 如果 1.13 XC 进场，则止损放在 1.27 XC + 少许空间

  2. 止盈：

    • 【斐波那契回撤】工具，从 C 拉到 D
    • 目标 1：0.5 CD
    • 目标 2：0.886 CD
    • 到达目标 1 后可以平仓一半，剩余仓位移动止损到成本价

鲨鱼可能会是最常见的和谐形态
因为有多个潜在反转区，要仔细观察市场对每个反转区的反应，再考虑是否进场
BC 延伸很多时候会超越 2.24（3.14 也有可能），但只要在 XC 0.886 - XC 1.13 之间都还可以接受

# 概念

5-0 模式是一种独特的结构，它拥有精确的斐波那契数列比对来验证结构。
虽然 5-0 被认为是一种回缩模式，因为 50% 的回缩是潜在反转区 (PRZ) 中最关键的数字，但不同价格支柱的测量值与蝙蝠或加特利略有不同。
5-0 属于 5 点和谐价格结构家族。
但是，其他模式所需的 B 点对齐并不适用于 5-0 模式。相反，5-0 要求最小的迅速和谐脉冲波至少为 1.618，但不超过 2.24 在 C 点区分设置。
此外，5-0 需要一个倒数 AB=CD 测量来定义模式的完成度。

这种模式的基本前提是，在一个相反的趋势完成后，确定不同的反应。
有效的 5-0 模式通常代表重大趋势逆转的第一次回调。
在许多情况下，结构的 AB 分支是一个延伸趋势的失败的最终波。
在艾略特波浪术语中，AB 分支可能是校正 “abc” 的失败波 3 或整个完成趋势的失败波 5。
虽然这些是明显的相似性，但从和谐交易的角度来看，重要的是通过其相关的斐波纳契测量来检查结构，以满足模式需求。
5-0 是一个非常精确的模式，它只包含两个数字：BC 支柱的 50% 回缩和 AB=CD 的倒数。
需要注意的是，用于确定潜在反转区 (PRZ）的测量方法与所有其他和谐模式有两种不同的方式：最后一段的结构完成点和 AB=CD 模式的计算。

虽然该模式在结构中包含了 5 个点 (X, A, B, C, D)，但是初始价格分支 (0) 可以从任何延伸价格移动的开始处开始。然而，初始点 (X) 必须具有相对于 a 和 B 点的特定对齐。
结构的 X、A、B 的形成通常是某种类型的极值和谐脉冲运动。
定义 B 点的 XA 投影最好不超过 2.24。
任何大于 2.24 的延伸都会抵消结构，因为较小的脉冲移动是首选。
再一次，这是失败的波 3 或波 5 在艾略特波术语 - 建立结构的其余部分。

  • 在 X、A、B 点寻找具有 1.13-1.618 延伸的初始脉冲反转
  • AB 投影必须在 C 点有最小 1.618 的延伸，但不超过 2.24
  • D 点由 50% BC 回撤和 AB=CD 模式完成定义

BC 支支柱是结构的最长价格长度，它必须至少是 AB 长度的 1.618 的延伸，但它必须不超过 2.24。
这个 1.618-2.24 的小范围是这个结构的定义元素。
如果没有达到 1.618 AB 的最小延伸，结构就不是有效的 5-0 模式。
在 BC 段从该区域反转后，从 B 点到 C 点测量 50% 的回缩。
此外，倒数 AB=CD 从 C 点 (AB 支柱的等效长度) 投影到潜在反转区（PRZ）上。

虽然 5-0 模式不是典型的 M 型或 W 型价格结构，但和谐交易方法中的相同原则也适用于这些情况。
倒数 AB=CD 有助于定义一般面积，而 50% 的回撤精确地确定了和谐区的精确范围。

下面的插图和示例将清楚地解释这些概念。

# 看涨的 5-0 模式

看涨 5-0 模式，如图所示，从 0 点开始，代表了从 X 点开始的向下延伸，初始点 (X) 作为之前大幅下跌的低点。
在快速反弹到 A 点后，结构突然继续下降，只是在 X 点找到稍高于之前低点的支撑。
这是失败的波 3 或波 5—— 在艾略特波术语中 —— 建立了结构的其余部分。

从和谐交易的角度来看，重要的的读数范围内要求这个 X 延伸至少 1.13，但不大于 1.618。
在这个脉冲失效波建立后，BC 分支至少反弹到 AB 长度的 1.618，但它不超过 2.24。

同样，这个 1.618-2.24 的小范围是这个结构的定义元素。
如果没有达到 1.618 的的读数范围内，结构就不是有效的 5-0 模式。
在 BC 线从该区域反转后，从 B 点到 C 点的牛市 50% 回档位被测量。

此外，倒数 AB=CD 从 C 点 (AB 支柱的等效长度) 投影到潜在反转区（PRZ）上。
识别这种结构需要一些时间，但最明显的特征是失败的下行波，然后是精确的 1.618-2.24 的延伸。
在这一点上，重要的是计算 50% 回撤水平的倒数 AB=CD 和研究价格在价格（PRZ）的作用。

# 看跌的 5-0 模式

如图所示，在 0 点开始，代表一段低的机会的初始点 X。
X 初始点的模式设置失败突破区域，那里的机会 B 从一个点到峰值，名义上再次拿出之前高 X，这是失败的一波 3 或一波 5 艾略特波浪条件建立了其余的结构。

请记住，此 XA 延伸名必须至少为 1.13 但不大于 1.618。
在脉冲失败波建立之后，BC 段下降到 AB 长度的至少 1.618 的延伸，但不超过 2.24。同样，这个 1.618-2.24 的狭窄范围是结构的一个决定性因素。如果未达到 1.618 的的读数范围内，则该结构不是有效的 5-0。在 BC 段从该区域反转后，从 B 点到 C 点测量看跌的 50% 回档位。

5-0 形态也称为鲨鱼之后的形态，因为大多数的鲨鱼完成之后可能形成 5-0 形态
也可以理解成鲨鱼达到目标 1 后的反弹，这个反弹可能小也可能大

# 规则

# 形态要素

反射 ABCD （Reciprocal ABCD）


# 形态确认

操作步骤：

  1. 先画出鲨鱼形态，等形态完成后
  2. 【趋势线】工具，从 B 到 C 画一条趋势线，并复制到 D 点
  3. 【斐波那契回撤】工具，从 C 到 D，寻找 E 点反转区
    • 可以是 CD 0.5，也可以是 CD 0.618
    • 如果反射 ABCD 低于 CD 0.5，则反转区定在 CD 0.5
    • 如果反射 ABCD 超越 CD 0.5，则反转区定在 CD 0.618

# 止盈止损

  1. 止损：

    • 如果进场在 CD 0.5，则止损在 CD 0.618 + 少许空间
    • 如果进场在 CD 0.618，则止损在 CD 0.786 + 少许空间

  2. 止盈：

    • 目标 1：DE 0.618
    • 目标 2：D 点

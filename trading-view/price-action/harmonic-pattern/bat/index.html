Bat 蝙蝠形态

首页 交易策略 价格行为 谐波形态

# 概念

  • 在整个和谐交易工具库中，蝙蝠形态或许是最精确的形态。
    该形态包含许多明确的元素，定义了极好的潜在反转区 (PRZ)。
    该形态通常代表着对支撑或阻力的再次深幅测试，这种测试常常非常迅速。
    从蝙蝠形态 PRZ 开始的迅速反转很常见。
    实际上，从蝙蝠形态开始的有效反转常常包含非常极端的价格行为。

  • 该形态包含强大的 0.886 XA 回调，是 PRZ 中的决定性元素。
    B 点回调必须小于 0.618，最好是 XA 臂的 0.50 或 0.382。

  • 最理想的 B 点比率是 XA 臂的 50%。
    B 点是区分蝙蝠形态和伽利形态的主要方法之一。
    如果一个形态正在形成，B 点刚好在 XA 臂的 0.50 处，那么它很可能是一个蝙蝠形态。

  • 蝙蝠形态使用的 BC 延伸至少是 1.618。
    BC 延伸可以长达 2.618。
    然而，蝙蝠形态中最理想的 BC 延伸是 1.618 或 2.0。
    需要注意的是，BC 延伸一定不能是 1.27，因为，如果 BC 延伸小于 1.618，那么该结构将失败。
    另外，1.27 BC 延伸通常见于伽利结构。

  • 蝙蝠形态中的 AB=CD 也是该结构有别于其他结构的要素。
    该形态通常是延长的，在理想情况下，蝙蝠结构中包含一个 1.27 AB=CD 计算结果。
    然而，等距 AB=CD 形态的确是令任何蝙蝠形态成为有效架构的最低要求。

  • 蝙蝠形态的准确性令人难以置信，所需止损远小于其他大部分形态。
    0.886 回调是 PRZ 中最重要的数字，但是，对于结构内的其他和谐数字而言，紧密收敛以限定一个精确的区间是共同的特征。

# 看涨蝙蝠形态

B 点低于 0.618 回调，最好是 XA 臂的 0.50 或 0.382，那么就定义了看涨蝙蝠形态。
在这一架构中，看涨 0.886 回调非常关键。
通常，看涨蝙蝠形态是一种极好的 5 点调整结构，出现于已经明确确定的支撑位。

# 看跌蝙蝠形态

B 点低于 0.618 回调，最好是 XA 臂的 0.50 或 0.382，那么就定义了看跌蝙蝠形态。
看跌蝙蝠形态是非常出色的 5 点调整结构，常常在回调至一个关键高点后形成。
另外，当这些结构形成于明显的阻力位时，它们是极好的交易机会。

0.886 XA 回调是看跌蝙蝠形态中最关键的价位。
由于这一回调的出现靠近前一高点 ——— 形态的初始起点 (X)，所以价格行为代表了对先前阻力的重大挑战。
蝙蝠形态表现出的结构性信号是一个重要的技术因素，其重要性高于其他简单的再次测试。
在明显阻力位形成的蝙蝠形态常常表明趋势中将出现实质性的改变 ——— 任何时间框架上都是如此！
等待这样明确的机会是非常重要的。
不用说，等待整个 PRZ 区间被测试，以确认形态的完成。
虽然这可能使交易的执行略有延迟，但是有效反转将在形态完成后不久提供决定性的信号。

# 规则

# 形态要素

  1. B 点低于 XA 的 0.618 回调，最好是明显的 0.5 或 0.382 回调
  2. BC 延伸至少是 1.618
  3. AB=CD 形态通常是延长的
  4. 0.886 XA 回调
  5. C 点在 0.382 和 0.886 构成的区间之内

# 形态确认

# 886 普通蝙蝠

  • 这形态最关键的斐波那契是 XA 的 0.886 回撤
  • B 点回调必须在 XA 的 0.382 - 0.5 之间，不能碰到 0.618
  • 和加特利形态一样，D 点需要 3 条斐波那契线重叠才算成立
  • 失败的蝙蝠可能变成 113 蝙蝠，也有可能变成螃蟹形态

操作步骤：

  1. 第一步是寻找可用的 X 点：X 点一般是明显的 V 或 U 高 / 低点（可能有多个，都测试一下）
  2. 【斐波那契回撤】工具，从 X 拉到 A，来寻找 B 点
    • 如果 B 点回撤到达 0.382 - 0.5 XA 之间，则为合格
    • 如果是正好到达 0.382 XA，很可能是变异蝙蝠
  3. 【斐波那契回撤】工具，从 A 拉到 B，来寻找 C 点
    • 如果 C 点回撤在 0.382 - 0.886 AB 之间，则为合格
  4. D 点由 3 条斐波那契线组成，这 3 条线重叠的区域，就是所谓的 D 点或称潜在反转区（PRZ）
    1. XA 0.886
      • 【斐波那契回撤】工具，从 X 拉到 A，得到 XA 回调
      • 在 0.886 XA 画水平线
    2. BC 1.618 - 2.618 （BC 没到 1.618 就还不成立）
      • 【斐波那契回撤】工具，从 B 拉到 C，得到 BC 延伸
      • 找到离 0.886XA 最近的斐波那契参数线，画出区间块
    3. AB = CD （一般都到 1.27 AB=CD）
      • 【斐波那契趋势扩展】工具，从 A 到 B 再到 C
      • 找到 1.27 的位置，对上述区间块进行微调





特殊情况：
XA 合格，出现两个回踩 B 点，如何选择？

应选择第二个出现的 B 点，因为第二个 B 点后有更高的 C 点，第二个 B 点位置稍低

如果蝙蝠的 C 点碰到 A 点，则要重新选择，此时的 C 点是新的 A 点，重新分析

# 113 变异蝙蝠

  • B 点位置在 0.382 XA 左右，小于 0.5 XA，可能完成蝙蝠后又反转，或可能直接顺趋势，去到更高的 D 点
  • 此时该 D 点的位置在 113，需警惕这样的情况

操作步骤：

  • 前面寻找 XABC 步骤相同，D 点确认步骤如下：
    1. XA 1.13
      • 【斐波那契回撤】工具，XA 1.13 画一条水平线
    2. BC 2.0 - 3.168
      • 【斐波那契回撤】工具，BC 延长，找到最接近上述水平线的斐波那契参数，画反转区间
    3. AB = CD （一般都到 1.618AB=CD）
      • 【斐波那契趋势扩展】工具，从 A 到 B 再到 C
      • 找到 1.618 的位置，对上述区间块进行微调



# 入场时机

# 886 普通蝙蝠

  • 最重要的是要碰到 XA 0.886 那条水平线，否则 886 蝙蝠的形态不成立
  • 可能不会测试整个 PRZ
  • 如果测试 PRZ 1-3 次可以接受，测试 3 次以上需小心形态失败
  • 查看 RSI 是否到达超买超卖，等 RSI 走出后，观察：
    • 反转蜡烛形态（最简单）
    • 打破小时间段趋势线（盈亏比较差）
    • 等蜡烛反转后，挂单在反转区进场（有可能错过）
    • 暴增量
  • 分批进场





# 113 变异蝙蝠



  • 当来到反转区，又配合暴增量，可以入场

普通蝙蝠（B 点 0.5XA）也有可能变成 113 蝙蝠
所以不管是交易蝙蝠还是 113 蝙蝠，如果价格达到 XA 0.886 不反转，都要关注 XA 1.13 的第二个潜在反转机会。
如果 BCD 都被快速完成，代表动力太强，很可能 886 和 113 都失败，而会突破去到螃蟹

# 止盈止损

  1. 【斐波那契回撤】工具，从 X 拉到 A

  2. 止损位置：一号止损位为 1.13 XA，二号止损位为 1.27 XA。
    一般放在二号位，防止碰到一号位被打止损后反转。

  3. 【斐波那契回撤】工具，从 A 拉到 D

  4. 止盈位置：一号止盈位为 0.382 AD，二号止盈位为 0.618 AD
    到达一号位后可以平仓一半，剩余仓位移动止损到成本价

  5. 止损位置：三号位为 1.13AD
    如果有出现暴增量，可以将止损放在 X 点以下，也就是 1.13AD 附近。

# 补充

仓位计算：

假设入场价格为 24.14，止损价格设为 21.74，则差价为 2.4，约 10%。

假设有本金 10000U，可以接受 5% 的亏损，即 500U。

500 / 2.4 = 仓量或合约量 约 208

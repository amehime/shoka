Gartley 加特利形态

# 概念

加特利形态的要素：

  • XA 臂精确的 61.8% B 点回调
  • BC 延伸一定不超过 1.618
  • 等距 AB=CD 形态是最常见的
  • 0.786 XA 回调
  • C 点在 0.382 ~ 0.886 回调构成的区间之内

根据经验，任何小于理想加特利斐波那契队列的结构，最终通常成为 “更深的” 调整结构，有利于蝙蝠形态的完成。
虽然这种对加特利队列的严格应用减少了形态匹配的数量，但关键是辨识出有效的形态，并且有效区分这两种类似的结构。

# 看涨加特利形态

一个明显的 AB=CD 形态和一个 0.786 回调，定义了看涨加特利形态中的潜在反转区 (PRZ)。
然而，这些条件仅在 XA 臂的 0.618B 点回调满足时才有效。

BC 延伸应该作为 PRZ 中其他数字的补充，1.13、1.27、1.41 或 1.618 延伸应该会聚于同一区域。
需要注意的是，加特利形态并不使用任何大于 1.618 的 BC 延伸。
任何大于 1.618 的 BC 延伸通常见于蝙蝠结构 。
需要指出的是，C 点的 “0.382；0.886” 斐波那契回调幅度可以是这两个限制比率之间的任何一个和谐交易比率。
因此，C 点可能是 0.382、0.50、0.618、0.707、0.786 或 0.886。

# 看跌加特利形态

看跌加特利结构主要是由 XA 臂的 0.618B 点回调来辨识。
形态的其他要素：AB=CD、BC 延伸和 0.786 回调，收敛时应该彼此接近，以便定义 PRZ。

# 完美看涨加特利形态

完美看涨加特利形态主要是由 C 点处精确的 0.618 回调来定义的。
虽然理想的价格行为应该测试潜在反转区 (PRZ) 的整个区间，但是最低要求 AB=CD 和 0.786 XA 臂仍然代表着完成形态中最重要的数字。
这种结构应该明显且具有理想的对称性。
另外，AB=CD 应该是完美的看涨结构，0.618 和 1.618 比率在相应的点处。
这些要素的结构定义了一种引人注目的理想架构，拥有特别重要的 PRZ。
虽然这些结构是稀有的，但是它们代表着所有加特利变种的最佳队列。

# 完美看跌加特利形态

完美看跌加特利形态由 B 点处一个精确的 0.618 回调来定义。
另外，完美看跌 AB=CD 和 0.786XA 臂必须得到 一个 1.618BC 延伸的补充。

加特利形态是一种有争议的形态。尽管《伟大的加特利辩论》 中有不同的解释，但是根据《和谐交易者》中最初描述的结构化规则，最和谐的加特利形态的所有点都具有明显的特征。
在定义有效的加特利结构方面，0.618B 点回调是关键的。
虽然 0.786 XA 回调是结构的一个重要元素，但是形态内的等距 AB=CD 才是潜在反转（PRZ）内最重要的完成点。
另外，AB=CD 的完成是所有有效加特利形态的强制性最低要求。

# 规则

# 形态要素

  1. 精确的 XA 臂的 0.618 B 点回调。
  2. PRZ 中 XA 臂精确的 0.786 D 点回调。
  3. 1.27 或 1.618 BC 延伸。
  4. 变种 AB=CD 形态。
  5. C 点回调可能在 0.382 和 0.886 之间变化。

加特利形态必须包括这些具体的条件才能定义最和谐的结构。
虽然加特利形态可能与蝙蝠形态相似，但是该架构中使用的斐波纳契比率是独一无二的。
另外，该形态应该相当明显，而且具有理想的对称性。

# 形态确认

  • 形态一共有 5 个点，X，A，B，C，D
  • 每一个点都必须符合一些规则，形态才算成立

操作步骤：

  1. 【斐波那契回撤】工具，从 X 拉到 A，得到 XA 回调
  2. B 点回踩的位置，位于 XA 回调的 0.618 的位置，则为合格
  3. 【斐波那契回撤】工具，从 A 拉到 B，得到 AB 延伸
  4. C 点回调需要满足在 AB 0.382 - 0.886 之间
  5. 寻找 D 点的位置，一般都是一个区域，是由 3 条斐波那契线组成
    1. 找到 AB = CD
      • 【趋势线】工具，从 A 画到 B
      • 复制一条，放在 C 点的位置
      • 在 D 点位置画一个方框
    2. XA 0.786
      • 【斐波那契回撤】工具，从 X 拉到 A，得到 XA 回调
      • 如果第一步所得 D 点没有超过 0.786 XA 回调，则反转区范围为 XA 0.786 ~ D 点
      • 如果第一步所得 D 点超过 0.786 XA 回调，则反转区范围为 D 点～XA 0.886
    3. BC 1.13 - 1.618
      • 【斐波那契回撤】工具，从 B 拉到 C，得到 BC 延伸
      • 反转区位于 BC 1.13 - 1.618 之间，微调 D 点框到最接近的 BC 延伸





# 入场时机

  • 一定要等待形态完成，形态完成是指完整测试了整个反转区，而不是刚刚触及反转区就入场。
    • 有可能会二次测试到更好位置，真正完成测试反弹才有力
    • 等走出反转区，再入场也不迟



  • RSI 超买超卖是辅助验证的加分项，不一定非要符合
    • 如果出现背离，也是加分项

加特利反转区最重要的两个点是 0.786 XA 和 0.886 XA

# 止盈止损

和 AB = CD 形态一样

  1. 【斐波那契回撤】工具，从 A 拉到 D
  2. 止损位置：一号止损位为 1.13AD，二号止损位为 1.27AD。（图中分别为 -0.13 和 - 0.272）
    一般放在二号位，或一号位下方一点，防止碰到一号位被打止损后反转。
  3. 止盈位置：一号止盈位为 0.382AD，二号止盈位为 0.618AD
    到达一号位后可以平仓一半，剩余仓位移动止损到成本价

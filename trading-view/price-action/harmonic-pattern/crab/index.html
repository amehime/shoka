Crab 螃蟹形态

# 概念

螃蟹形态是一种明显的 5 点延伸结构，专门使用 XA 臂的 1.618 延伸。
这是该形态最重要的一点，是潜在反转区 (PRZ) 中的决定性价位。
BC 的极端 (2.618、3.14、3.618) 延伸是 1.618XA 延伸的补充。
另外，螃蟹形态主要利用变种 AB=CD 来补充 PRZ。
虽然最低 AB=CD 完成对于有效结构是必需的，但是变种 1.27 或 1.618 是 PRZ 中最常见的。
实际上，1.618 AB=CD 形态是该结构中使用最常见的变种。

需要注意的是，螃蟹形态内的变种 AB=CD 形态是 PRZ 中 最不重要的数字。
XA 臂的 1.618 延伸和极端 BC 延伸的结合，通常会为形态的完成制造一个明确的区间。
实际上，与其他所有形态相比，AB=CD 计算或许在螃蟹形态中是最不重要的。
虽然 AB=CD 结构在螃蟹形态中不那么重要，但是该形态仍然应该具有明显的对称性。

# 看涨螃蟹形态

看涨螃蟹形态具有明显的结构，最后一条臂特别长。
随着该形态接近其完成点，价格行为常常变得非常极端。
另外，最好等待 1.618XA 延伸被测试，但是 BC 延伸应该要重点考虑，而且是 PRZ 内最低价位。
此外，变种 1.27 或 1.618 AB=CD 应该补充这一区域。

# 看跌螃蟹形态

看跌螃蟹形态是一种精确的结构，它的 PRZ 包含三个数字：1.618XA 回调、变种 AB=CD 和极端 BC 延伸。

XA 和 BC 延伸的会聚定义了潜在反转区的顶部区间。
1.27 和 1.618 变种 AB=CD 形态通常补充形态完成点处的其他数字。
正如前面提到过的，对于螃蟹形态来说，AB=CD 形态在 PRZ 中不像其他形态那么重要。
最低限度，XA 必须在反转前被测试，然后才能认为形态完成，才能认为它是一个有效的架构。
在初次测试 PRZ 时，虽然价格行为可能略微超越 1.618XA 臂，但是有效螃蟹形态中的完成点会标记出精确的反转点。

# 看涨深海蟹形

看涨深海蟹形是一种不同的结构，由 XA 臂的 0.886 价格 B 点回调来定义。
1.618 价格 XA 臂应该是 PRZ 中的最低点，是形态完成的决定性限制。
该形态通常包含某种类型的变种看涨 AB=CD，需要 1.27 或 1.618 延伸。

# 看跌深海蟹形

深海看跌蟹形专门使用一个 1.618XA 延伸来作为 PRZ 中 的决定性价位。
B 点处 0.886 回调使得该结构具有了 “深海” 特征。
从本质上讲，这个形态包含多重延伸，与变种 AB=CD 结构相结合，辨识出相当可观的潜在反转点。
该形态常常形成于趋势已经在带有极端 BC 延伸的形态中确立之后。

虽然螃蟹形态的结构化变种可能看起来定义了相同的形态，但是斐波那契比率队列清楚地把每种类型区分为独一无二的形态。
螃蟹形态通常确定的是极端的超买和超卖行情，因为 1.618XA 臂是该形态最重要的斐波那契比率尺寸，它代表着潜在反转区 (PRZ) 中最重要的数字。
与和谐交易法中的其他形态不同，AB=CD 在完成点处不是那么关键。
虽然为每个变种区分 AB=CD 非常重要，但是变种 1.618 AB=CD 形态是大多数螃蟹形态中的首选结构。
然而， 1.27AB=CD 形态通常出现于原始版本中，而深海蟹形包含一个等距 AB=CD 也并非不常见。
极端 BC 延伸 ——— 通常是 2.618、3.14 或 3.618 延伸 ——— 突出这种结构的特征。
虽然深海蟹形允许 BC 有较宽泛的变化，但是这一延伸应该与 1.618XA 紧密地会聚。

# 规则

# 形态要素

  1. B 点是 XA 0.618 回调或小于 XA 0.618 回调
    • XA 臂精确的 0.618 B 点回调或更短的回调 (0.382 和 0.50 是该结构中常见的 B 点)。
  2. 极端 BC 延伸，通常是 2.618、3.14 或 3.618
    • 有时该结构中会形成 2.24 BC 延伸
  3. 最低 AB=CD 完成。变种 1.27 AB=CD 或 1.618 AB=CD 形态是最常见的。
  4. 1.618 XA 延伸作为该结构的决定性限制
  5. C 点回调可能在 0.382 和 0.886 之间变化。

这是一种极端形态必须强调这种形态易变的特征。
从这些 PRZ 开始的反转可能是迅速的，包含极端的价格行为。
螃蟹形态是一种独特的形态，拥有独特的性质，能够辨识出非凡的和谐机会。
这种形态的区别在定义每个变种时很关键。

从一个和谐的角度来看，有效的螃蟹结构辨识的是极端的超卖或超买行情。
在初次测试潜在反转区时，这些形态通常提供某种类型的反应。
价格行为常常响应极端的斐波那契延伸组合，在螃蟹形态内指的是 1.618XA 臂和 2.618 或更长 的 BC 臂的组合。
螃蟹形态拥有难以置信的准确度，是和谐交易工具库中一种稳定有效的形态，能够辨识出市场中的关键反转点。

# 形态确认

操作步骤：

  1. 寻找可用的 X 点，X 点一般是明显的 V 或 U 形的高点或低点
  2. 【斐波那契回撤】工具，从 X 到 A，寻找 B 点
    • 如果 B 点回撤到达 XA 0.382 - 0.618 （螃蟹） 或 0.886（深蟹），则合格
  3. 【斐波那契回撤】工具，从 A 到 B，寻找 C 点
    • 如果 C 点回撤在 AB 0.382 - 0.886 之间，则合格
  4. D 点由 3 条线组成：
    1. XA 1.618 - 1.902
      • 【斐波那契回撤】工具，从 X 拉到 A，得到 XA 回调
      • 在 1.618 XA 和 1.902 XA 画两条水平线
      • 常见的螃蟹都会在 XA 1.618 小幅度反转，然后二次测试反转区；然后甚至可能会到达 XA 1.902 完成一个大反转
    2. BC 2.618 - 3.618
      • 【斐波那契回撤】工具，从 B 拉到 C，得到 BC 延伸
      • 在 BC 2.618 - 3.618 的区间中，找到最接近上述两条水平线的斐波那契参数，画两个区间
      • BC 延伸不是关键，即使达到 BC 4.23 的螃蟹都一样有效，不必太在意是否超越书里写的 3.618 极限。
    3. 1.618 AB = CD
      • 【斐波那契趋势扩展】工具，从 A 到 B 再到 C
      • 找到至少 1.618 的位置，对上述区间块就近找参数进行微调

螃蟹一般发生在趋势的结尾，而且反应起来的速度非常快，所以判断后进场要快，否则错失机会。





# 入场时机

如果熊的趋势很强，注意观察盈亏比是否值得现价入场。如果盈亏比不好，则等待回踩 PRZ，如果错过就不再入场。

# 止盈止损

  1. 【斐波那契回撤】工具，从 X 拉到 A

  2. 止损位置：1.618 XA 入场，则止损位放在 1.902 XA 以上一点点

  3. 【斐波那契回撤】工具，从 A 拉到 D

  4. 止盈位置：一号止盈位为 0.382 AD，二号止盈位为 0.618 AD
    到达一号位后可以平仓一半，剩余仓位移动止损到成本价

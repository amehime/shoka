Butterfly 蝴蝶形态

# 概念

蝴蝶形态，是一个可以产生反转信号的重要形态。
价格在经历了蝴蝶形态发生反转后，往往会产生强劲的运动。
它是一个非常强大的和谐交易形态。
蝴蝶形态的基本结构，由 Bryce Gilmore 发现。但是，我在对于该形态的结构定义中，引入了一些重要元素，以而定义了理想蝴蝶形态的结构。
对于蝴蝶形态来说，其中最重要的一个极成条件，即是其中 AB=CD 形态经常会采用其拉长延展的形态。

# 看涨蝴蝶形态

从一个精确的 0.786 B 点回调开始，这个看涨蝴蝶形态包含一个最低要求的 AB=CD 形态，1.27XA 延伸和看涨蝴蝶形态 1.618 BC 延伸是定义潜在反转区 (PRZ) 的强势性要求。

# 看跌蝴蝶形态

理想看跌蝴蝶结构是由 XA 臂的精确的 0.786B 点回调定义的。
该架构需要一个 AB=CD 形态，应该与 1.27 XA 延伸和 1.618 BC 延伸会聚在一起，这是定义 PRZ 的最低条件。

# 完美看涨蝴蝶形态

完美看涨蝴蝶形态是一种明显的 5 点延伸结构，需要具体的斐波那契数字队列，包括 PRZ 中的最低限制 1.27XA 延伸。
该形态也必须包含一个带精确 1.618 BC 延伸的 1.27 AB=CD 形态。

# 完美看跌蝴蝶形态

完美看跌蝴蝶形态是一种明显的 5 点延伸结构，利用 1.27 XA 延伸作为 PRZ 中的最高限制。
该形态应该具有一个带 1.618 BC 延伸的变种 1.27 AB=CD 形态补充 PRZ。

蝴蝶形态是一种特殊的 5 点延伸结构，具有独特的斐波那契比率队列来确认该形态有效。
该形态的首要决定性元素是 XA 臂的精确的 0.786 B 点回调。
蝴蝶形态只使用一个 1.27 XA 延伸作为 PRZ 中最重要的数字。
此外，等距 AB=CD 形态是该形态完成的最低要求。
但是，对于该结构来说，变种 1.27 AB=CD 形态更为常见。
BC 延伸是这种形态的又一个决定性元素，它必须至少是一个 1.618 延伸。
然而，蝴蝶形态中 BC 延伸不像在螃蟹形态中那么极端。

# 规则

# 形态要素

  1. B 点回调必须精准到达 XA 的 0.786 的位置。
  2. C 点回调可能在 AB 0.382 和 0.886 之间变化，不 0.618 是首选。
  3. BC 延伸可能在 1.618 和 2.618 之间变化，具体取决于 C 点回调。
  4. 反转区间必须完成 AB=CD 形态，变种 1.27 AB=CD 最为常见，一般都会达到。
  5. D 点和之前的形态一样，由 3 条斐波那契线重叠组成

D 点的参数在 2019 年有被优化，之前 D 点只接受 AX 1.27，现在则能接受 1.27 - 1.414
BC 延伸找 D 点也从之前的 1.618 -2.24 变成 1.618 - 2.618

蝴蝶形态类似于加特利形态，因为它需要一个特殊的斐波那契比率队列 ——— 特别是一个精确的 B 点 ——— 来确认这种结构有效。
虽然如此严格的定义减少了可能的交易机会的数量，但是这种精确性减少了无效的情形。
那些无效的情形可能看起来相似，但却不能产生一致准确的架构。

D 点会超过 X 点，一般是发生在一些趋势的尾声。

# 形态确认

操作步骤：

  1. 寻找可用的 X 点，X 点一般是明显的 V 或 U 形的高点或低点
  2. 【斐波那契回撤】工具，从 X 到 A，寻找 B 点
    • 如果 B 点回撤到达 XA 0.786 左右，则合格
  3. 【斐波那契回撤】工具，从 A 到 B，寻找 C 点
    • 如果 C 点回撤在 AB 0.382 - 0.886 之间，则合格
  4. D 点由 3 条线组成：
    1. XA 1.272 - 1.414
      • 【斐波那契回撤】工具，从 X 拉到 A，得到 XA 回调
      • 在 1.272 XA 和 1.414 XA 画两条水平线
      • 如果 1.272 XA 到了没反应，则有可能去到 1.414 XA
    2. BC 1.618 - 2.618
      • 【斐波那契回撤】工具，从 B 拉到 C，得到 BC 延伸
      • 在 BC 1.618 - 2.618 的区间中，找到最接近上述两条水平线的斐波那契参数，画两个区间
    3. 1.27 AB = CD
      • 【斐波那契趋势扩展】工具，从 A 到 B 再到 C
      • 找到至少 1.27 的位置，对上述区间块进行微调（有重叠则不调整；看 1.272 XA 如果靠近 1.414 AB=CD ，则将框拉向 1.414，否则拉向 1.27，就近调整；如果 1.414 XA 接近 1.618，则也需调整）





最关键的是 B 点要准确到达 0.786
XA 延伸必须在 1.272 - 1.414 之间
BC 延伸和 ABCD 所在位其实没有那么讲究

# 入场时机

大致同前
可以以首次和二次测试反转区，有反应后进场

# 止盈止损

  1. 【斐波那契回撤】工具，从 X 拉到 A

  2. 止损位置：一号止损位为 1.414 XA，二号止损位为 1.618 XA。
    当 1.272 XA 入场时，则止损在 1.414 XA，在 1.414 XA 入场，则止损在 1.618 XA

  3. 【斐波那契回撤】工具，从 A 拉到 D

  4. 止盈位置：一号止盈位为 0.382 AD，二号止盈位为 0.618 AD
    到达一号位后可以平仓一半，剩余仓位移动止损到成本价

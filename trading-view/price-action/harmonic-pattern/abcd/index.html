AB=CD 形态

# 概念

AB=CD 形态是一种四点价格结构，初始价格波段被部分回撤，回撤完成后接着一波与回撤等幅的运动。
这种结构已经在以前很多技术著作中讨论过，它代表了所有和谐形态的基础。
形态中具体的点应满足斐波那契比率。
在 AB=CD 形态中，C 点将是确认形态完成的一个关键点。
在这种结构中，虽然 BC 延伸非常重要，但是最重要的和谐数字是 AB=CD 的确切完成点。

# AB=CD 倒数比率

在 AB=CD 形态中，结构内的斐波那契比率队列通常表现出特殊的倒数关系。
AB 臂的 C 点回调的倒数比例通常显示哪一个 BC 延伸被用于定义潜在反转区 (PRZ)。

举例说明，C 点处的 0.618 回调将具有一个 1.618BC 延伸，在最接近 AB=CD 形态的完成处收敛。
等距 AB=CD 形态内的这种倒数关系为该结构定义了最佳的 PRZ。

# 等距看涨 AB=CD 形态

当观察明确的抛售行情时，看涨 AB=CD 形态是一种极好的度量工具。
虽然对称性可能改变，但是这种基本结构是所有和谐形态的最低要求。

需要指出的是，C 点的 “0.382；0.886” 斐波那契回调幅度可以是这两个限制比率之间的任何和谐交易比率。
因此，C 点可能是 0.382、0.50、0.618、0.707、0.786 或 0.886。
参考上面列出的倒数比率，相对应的 BC 延伸可能是 1.13、1.27、1.41、1.618、2.0、2.24 或 2.618。在某些罕见的情况下，也可能用到 3.14 延伸。

# 等距看跌 AB=CD 形态

看跌 AB=CD 应该包含一个明显的对称，形态的完成点补充 BC 延伸，定义一个精确的 PRZ


需要指出的是，C 点的 “0.382；0.886” 斐波那契回调幅度可以是这两个限制比率之间的任意一个和谐交易比率。
因此，C 点 可 能 是 0.382、0.50、0.618、0.707、0.786 或 0.886。
参考倒数比率，相对应的 BC 延伸可能是 1.13、1.27、1.41、1.618、2.0、2.24 或 2.618。在某些罕见的情况下，也可能用到 3.14 延伸。

# 变种看涨 AB=CD 形态

变种看涨 AB=CD 形态通常形成于长期抛售行情中类似看涨螃蟹形态和看涨蝴蝶形态这样的形态中。
然而，相对于 PRZ 中的其他斐波那契数字，变种 AB=CD 形态只是一个补充元素。

另外，与变种结构相比，等距 AB=CD 形态通常包含更重要的完成点。
1.618 AB=CD 形态的使用频率低于其他 AB=CD 组合。
尽管存在差异，但是当应用于校正和谐形态时，每个 AB=CD 元素都能够定义精确的反转区，提供有效的方法来量化价格结构。

# 变种看跌 AB=CD 形态

需要注意的是，AB=CD 形态 ——— 等距或变种 ——— 是所有和谐结构的基础。
大多数结构中，等距 AB=CD 形态是入场的最低要求。当与其他重要的斐波那契回调和延伸结合使用时 AB=CD 形态能够确定极好的反转区。

虽然基本的 AB=CD 结构可能包含各种各样的斐波那契比率，但是在两条连续的明显价格臂完成点处存在支撑或阻力的思想，是所有和谐形态的本质。

变种 AB=CD 形态强调了使用基本结构来定义具体形态完成的重要性。
无论哪种 AB=CD 形态，BC 延伸都应该是形态完成点的补充元素。
重要的是，要记住 C 点与 BC 延伸的倒数关系。
完美 AB=CD 形态使用 0.618 回调和 1.618 延伸，对于该形态，它们是最和谐的斐波那契比率队列。这一形态应该拥有具体的特征，使它成为一个有效的和谐结构，并且定义一个交易机会。

# 规则

# 形态要素

  1. AB=CD 完成的最低要求是每条价格臂相等。
  2. C 点回调可能在 0.382 和 0.886 之间变化，不过 0.618 是首选。
  3. BC 延伸可能在 1.13 和 2.618 之间变化，具体取决于 C 点回调。
  4. 变种 AB=CD 形态存在。

# 形态确认

操作步骤：

  1. 【斐波那契回撤】工具，从 A 到 B 画斐波那契，得到 AB 延申
  2. 如 C 点回调到 AB 延申的 0.382，未超过 AB 延申的 0.886，则满足形态要求，提示有潜在的 AB=CD 形态
  3. 【趋势线】工具，从 A 到 B 画条趋势线
  4. 【复制】AB 线到 C 点来寻找 D 点
  5. 【斐波那契回撤】工具，从 B 到 C 画斐波那契，得到 BC 延申
  6. BC 延申的斐波那契参数只要和 D 点重叠，完成 ABCD 形态的所有步骤了
  7. D 点与最近的 BC 斐波那契参数形成反转区



# 入场时机

到达反转区，形态完成，需要等 RSI 等指标过滤后再入场，勿提前挂单

  • RSI 走出 30 超卖 + 上升 K + 走出反转区 = 多
  • RSI 走出 70 超买 + 下跌 K + 走出反转区 = 空
  • RSI 超买超卖是辅助验证的加分项，不一定非要符合；如果出现背离，也是加分项

# 止盈止损

  1. 【斐波那契回撤】工具，从 A 拉到 D
  2. 止损位置：一号止损位为 1.13AD，二号止损位为 1.27AD。（图中分别为 -0.13 和 - 0.272）
    一般放在二号位，或一号位下方一点，防止碰到一号位被打止损后反转。
  3. 止盈位置：一号止盈位为 0.382AD，二号止盈位为 0.618AD
    到达一号位后可以平仓一半，剩余仓位移动止损到成本价

蜡烛图使用一条水平线来表示开盘价和收盘价，并用一个垂直框连接这两条线以形成烛台的 “主体”。

在盒子上方的中间绘制一条垂直线以显示最高价，在盒子下方的中间绘制一条垂直线以显示最低价。这些被称为 “灯芯” 或 “阴影”。

如果收盘价高于开盘价，则实体为空心（白色或绿色），表示当天价格走势向上。

如果收盘价低于开盘价，则实体关闭（黑色或红色）以指示下跌。

中国大陆、日本、韩国及台湾采用 “红升绿跌”，欧美及香港采用 “绿升红跌”

# 八种形态

  • 形态1️⃣ 买方力量很强
  • 形态2️⃣ 买方 大于 卖方；上影线代表卖，剩余代表买方
  • 形态3️⃣ 代表有强力卖方出现
    • 低点出现 ⇨ 锤形线
    • 高点出现 ⇨ 吊颈线
  • 形态4️⃣
    • 低点出现 ⇨ 倒锤形
    • 高点出现 ⇨ 射击之星
  • 形态7️⃣ 十字星
    • 低点出现 ⇨ 早晨之星
    • 高点出现 ⇨ 黄昏之星
  • 形态3️⃣ - 7️⃣ 代表 买方力量不够强， 需要看第二天的情况，如果第二天开高，则买方强势
  • 形态8️⃣ 可能人很少，量很小

总之就是变盘信号


# Hammer 锤形线

# 什么是锤形线

锤形线
是指锤形线组合的核心是一个下影线特别长，实体特别短小的 K 线，当这样的 K 线出现在下跌趋势中，如图所示。组合中的非关键 K 线都画成竖线，表示该位置上的 K 线形态不重要。

# 锤形线形态特征

  • 出现在下降途中，小实体在 K 线的顶部。
  • 下影线的长度一般至少是实体长度的 2～3 倍。
  • 上影线非常短，甚至没有。
  • 小实体可以是阴线，也可以是阳线，并不重要。
  • 锤形线处于下降趋势中。

# 锤形线的技术分析

锤形线出现在下降趋势中，当天的庞大卖压被有效遏制，买方在激烈的争夺中占了上风，以比较高的价格收盘。

此时，不少投资者担心踏空。

如果收盘价高于开盘价，称为一根阳线，行情会更有利于上涨。

第二天较高的开盘价和更高的收盘价将使锤形线的反转意义得到确认。

# 锤形线的操作要点

  • 如果锤形线的当日为跳空开盘，那么反转的意义较浓。
  • 一般在出现锤形线时，要注意当日的交易量，放量越明显，反转信号越可信。
  • 锤形线是比较次要的底部反转信号，出现后，必须等待次日的验证信号。如果信号确认买入后，投资者有必要在锤形线的底部设置止损位，一旦股价跌破该价位，则说明反转失败，应该及时止损出局。

# Hanging Man 吊颈线

# 什么是吊颈线

吊颈线是空头 K 线形态之一。

如当天大盘或个股股价平开盘或小幅跳高、跳低开盘，其后大幅下跌，但尾盘又大幅收高，留下长长的下影线，而 K 线实体无论是阳线或阴线都相当小，仅为影线的三分之一或四分之一，这样的 K 线即是市场中人所称的 "吊颈线"。

在这样的 K 线组合中，由于 “吊颈线” 是出现在高位，如同一根无钩的钓鱼线挂在高空，就好像姜太公钓鱼，愿者上钩，此时作多，无异于自投罗网，所说的 “姜太公”，当然是指最近一直在其中频密运作、在当天把股价拉回到高位的主力机构，而那些自以为是 “逢低吸纳” 的作多股友，就是那种连无钩的钓鱼线都能把它钓上来的笨鱼。

一般说来，如果出现这样的股谱，有货的投资者都应该在当中尾市拉高时及早清货，至少，在第二天跳空低开后，就应该毫不犹豫地出清手上持有的股票。

吊颈线与黄昏之星一样，都不是从单个 K 线便能得出短期或中期头部的可靠判断，吊颈线或黄昏之星都需几根 K 线的组合，才能形成较稳定的形态以判断后市股指或股价趋势的空头走向。

如吊颈线出现之前有加速上涨的放量长阳线，吊颈线出现之后马上跟随着中阴或长阴线，则后市出现下跌的趋势较明朗。

# 吊颈线的形成过程

股票在当天的相对高位开盘后无力继续上冲，出现较大幅度的回调，但尾市又得到买盘的支持或主力的拉抬而收于开盘位附近，从而形成一根 “T” 字形 K 线。这根 K 线的形态，非常象一个人用绳子把自己吊在一根横梁上自杀一样，因而有 “吊颈线” 之称。

而高位 “吊颈线” 则总是出现在股票持续攀升了一段时间之后。当股价最后加速上冲，终于有一天，在以较大的幅度开盘后，由于总体升幅太大，买盘后援不足，同时获利盘看到其高位滞涨，上冲动力减弱，获利了结的欲望也开始增强，加上主力也加大了出货的力度，于是卖盘增加，买盘却收手，造成接盘无力，股价走低，从而刺激更多的卖压出现，股价加速回调，有时甚至下跌到前一天的收盘价之下。

此后，由于短时跌幅较大，而市场情绪仍然比较乐观，一些原来看好该股却不敢追高买进的投资者此时总算等到了 “逢低吸纳” 的 “良机”，抓紧时机入市捡货，主力也停止卖出，反而顺势拉高，于是股价回到开盘价附近报收。

但由于此时已处高位，主力入市拉抬价格，通常并不是想要继续往上推升股价，而是为了方便第二天继续出货。在这种情况下，此股第二天多半无法继续上攻，相反，它们往往是向下跳空低开，而且开盘位多半是当天的最高价，开盘之后即开始逐步走低，最后总是以低于前收盘的价位报收。

这种情况，特别容易出现在连续涨停之后。在大多数情况下，收出 “吊颈线” 的当天，股价都不能以涨停报收，当然，也有少量的品种还是能继续涨停。但无论是什么情况，高位抛压加大这一点都是无法否认的。

# Inverted hammer 倒锤形

# 什么是倒锤形

倒锤形
形态与锤形线相反，倒锤形的核心是一个上影线特别长、实体特别小的 K 线，如图所示，该形态处于下降趋势中。

# 倒锤形的形态特征

  • 小实体在价格区域的较低部分形成。
  • 倒锤形在连续的下降趋势之后形成，形成的当天一般会向下跳空开盘。
  • 上影线一般比实体长两倍以上。
  • 下影线很短，有时甚至短到可以认为没有。

# 倒锤形的技术分析

持续的下降趋势使当日开盘股价向下跳空，当天上冲失败以较低价位收盘，形成一条很长的上影线。这种形态决定趋势反转是成功还是失败的关键是第二天的开盘，如果第二天开盘高于倒锤形的实体，潜在的趋势反转将引起空头的恐慌，这会支持股价进一步上涨，是股价发生反转的信号。

# 倒锤形的意义

  • 倒锤形出现的前一根 K 线为大阴线，且与倒锤形的实体部分形成跳空缺口者，分析意义甚大；
  • 倒锤形上影线较长，而下影线甚短，最好没有下影线，则为标准的形态；
  • 倒锤形之后的 K 线形态与倒锤形实体部分形成跳空缺口，形态的底部反转信号更加强烈；
  • 注意成交量的萎缩，暗示了底部杀跌力量的衰竭，抄底买盘即将进驻。

# 倒锤形的陷阱

  • 倒锤形是次要的底部反转信号，通常表现为下跌趋势中的弱势反弹。除了要等待第二日的验证信号外，还要预先设立止损位，倒锤子线的最低价就是设立止损位的一个很好参考；
  • 倒锤形次日的量无法跟上，无法收阳就可能是陷阱；
  • 我们看到倒锤头出现后，一定要符合买入条件，再杀入抢短线，否则有被套危险：底部见到倒锤头，上影线长喜心头；符合条件再买入，宜作短线别久留。

# 倒锤形的操作要点

  • 倒锤形必须出现在下降趋势之后，才具有看涨的可能。
  • 如果倒锤形出现后，第二天股价向上跳空开盘或在较高的价位上拉出一根阳线，则其转势向上的信号比较强，反转形态得到确认。
  • 倒锤形是比较次要的底部反转信号，因此其出现后，等待次日的验证信号显得十分重要。投资者应当把出现倒锤形那一天的最低价设定为止损位，一旦股价跌破该价位，则说明下跌支撑失败，投资者应及时止损出局。

# 倒锤形的操作策略

  1. 上涨趋势初期出现 “倒锤形” 底部峰值大于任何上档峰值时，积极作多；

  2. 上涨趋势中的循环底点出现 “倒锤形”，正逢筹码密集峰区域 — 果断介入；

  3. 下跌趋势中出现 “倒锤形”，在作多的市场里即便见底反弹，意义并不很大，上峰抛压犹存；

  4. 长期下跌末端出现 “倒锤形”，有见底的含义；

  5. 大幅上涨末端出现 “倒锤形”，恰遇底部筹码出兵向上，随其减仓操作或离场观望。

# Shooting Star 射击之星

# 什么是射击之星

射击之星又称为 “倒转锤头”，仿如枪的准头，是以有此称谓。

射击之星可以是阴线或阳线，但实体比较短小，上影线较长，其位置主要出现在某只个股的顶部，是一种十分明显的见顶信号。

这一形态的形成是开盘价比较低，多头组织力量向上攻，一度急升，但尾市卖压加强，收市价又回落至开盘价附近。

射击之星因为光芒短暂又被称为 “流星”。

射击之星通常出现在顶部，当大市已有一段升幅之后，一旦出现射击之星，往往预示着大市可能反转，准确性较高。

射击之星也会出现在底部，主要是投资人的心态不稳定所造成的。

因此，射击之星出现在底部是良好的买点，而出现在顶部则是良好的卖点。

# 射击之星的形态特征

  1. 此前存在一个明显的上升趋势。
  2. 在股价上升过程中出现一根带上影线的小实体，并向上跳空高开。
  3. 上影线的长度至少是实体长度的两倍；下影线短到可以认为不存在。
  4. 射击之星形态如果在高位出现，行情下跌的可能性较大。

# Morning Star 早晨之星

# 什么是早晨之星

早晨之星又称 “黎明之星”、“晨星”、“希望之星”，是由三根 K 线组成的 K 线组合形态，它是一种行情见底转势的形态。

这种形态如果出现在下降趋势中应引起注意，因为此时趋势已发出比较明确的反转信号，是一个非常好的买入时机。

早晨之星的 K 线形式一般出现在下降趋势的末端，是一个较强烈的趋势反转信号，谨慎的投资者可以结合成交量和其它指标分析，得出相应的投资参考。

# 早晨之星的形态特征

  1. 在下降趋势中某一天出现一根抛压强劲的长阴实体，显示短期走势可能会仍然向下，跌势可能会继续。
  2. 第二天出现一根向下跳空低开的十字型或锤型，且最高价可能低于第一天的最低价，与第一天的阴线之间产生一个缺口，显示跌幅及波幅已略有收缩，带来可能转好信号。具体的第二根 K 线的位置有时会不同，需要灵活的把握。
  3. 第三天出现一根长阳实体，买盘强劲，显示市况已转好，逐步收复失地。

# Evening Star 黄昏之星

# 什么是黄昏之星

黄昏之星又称 “暮星”，是一种类似早晨之星的 K 线组合形式，可以认为是后者的翻转形式，因此黄昏之星在 K 线图中出现的位置也与后者完全不同。

黄昏之星的情况同早晨之星正好相反，它是较强烈的上升趋势中出现反转的信号。

黄昏之星的 K 线组合形态如果出现在上升趋势中应引起注意，因为此时趋势已发出比较明确的反转信号或中短期的回调信号，对于我们来说可能是非常好的卖出时机或中短线回避的时机。

同时如能结合成交量的研判，对于提高判断的准确性有更好的帮助。

下面是黄昏之星在实际 K 线图中的走势情况，可以看出在走出形态组合的同时，成交量也放出，其后该股走出一段调整走势。

# 黄昏之星的形态特征

黄昏之星是由三支阴阳烛组成的转向利淡形态，通常在一个上升趋势后出现。

  1. 第一支烛为承接前期上升走势的大阳烛，买盘强劲，显示升势持续。
  2. 第二支烛可为出现在裂口高开后的十字星或纺锤。此讯号显示买方压力逐步得以舒缓，价格大有可能已见顶。倘若第二支烛有着与射击之星相同的上影线，利淡转向讯号的可靠性更大为提高。
  3. 第三支烛为沽盘强劲的阴烛，必须深入到第一根阳线中心以下。此时市况已发生根本的转变，跌势一直持续到收市。

# 什么是布林通道？

布林通道（Bollinger Band）是由金融分析师和交易员 John Bollinger 于 20 世纪 80 年代初创建的。布林通道作为（金融分析）工具被广泛应用于技术分析（TA）领域，从基本上来说，它是一种振荡测量器，用于表明市场中的高低波动，以及超买或超卖的情况。

布林通道的主要原理是强调价格是如何围绕平均值波动的。更具体一些来说，指示器由上轨，下轨和中间移动平均线（也称为中轨线）组成。两个横向波带对市场价格行为作出反应，当波动率较高时扩展（远离中轨线运行）而波动率较低时收缩（向中轨线运行）。

标准布林线公式将中轨线设置为 20 天简单移动平均线（SMA），而上轨和下轨则基于与 SMA 相关的市场波动率（称为标准差）计算。 布林线指标的标准配置如下所示：

  • 中轨线 = 20 日的移动平均线（SMA) = 平均成本
  • 上轨线 = 20 日的 SMA +（20 日的标准差 x 2）= 股价的压力线
  • 下轨线 = 20 日的 SMA -（20 日的标准差 x 2）= 股价的支撑线
  • 带宽 (通道空间) = (上轨−下轨)/ 中轨

标准布林线以 20 天为周期，将上下波带设置为远离中轨线的两个标准差（x2）。这样做是为了确保至少 85％的价格数据将在这两个波带之间波动，但也可以根据不同的需求和交易策略来调整设置规模。

# 如何使用「布林通道」评估进、出场时机

布林通道的概念，是认为价格有很大机率落在通道中间附近，因此衍生出 3 种多头讯号、2 种空头讯号：

# 多头讯号

  1. K 线自下而上攻击，穿越下轨线时是买进讯号。
  2. K 线自下而上攻击，穿越中间线时股价可能加速向上，是加码买进讯号。
  3. K 线落在中轨与上轨之间，且仍为多头向上趋势时，可继续做多。

# 空头讯号

  1. K 线落在中轨与上轨之间，并自上而下掼破中轨，是卖出讯号。
  2. K 线落在中轨与下轨之间，且仍为空头向上趋势时，可继续做空。

# 如何观察「布林通道」的型态来执行进、出场

布林通道的形态考量，就是考虑布林通道三条轨道线所构成的不同型态，基本上共分成：同向和异向 2 种主要型态。

  • 同向型态包括：
    • 三轨向上移动
    • 三轨向下移动
    • 三轨水准间运行
  • 异向型态包括：
    • 三轨张口
    • 三轨收口
    • 张口型喇叭口
    • 收口型喇叭口
    • 紧口型喇叭口

# 同向型态

# 三轨向上移动

布林通道同时上移且在一定时间内维持，这一般是预示多头力量稳健，盘面将稳定上涨，且期间回档较浅。

# 三轨向下移动

布林通道同时下移且在一定时间内维持，预示空头力量较为稳健，盘面逐步下跌，反弹也会较弱。

# 三轨水准间运行

布林通道水准指向最简单，参数为 (20,2) 时，这表示盘面将进行狭幅震荡，盘面连区间波段交易的机会都几乎没有，因为获利空间太小，故此种情况下，只能以观望盘势为主。

# 异向型态

# 三轨张口

布林通道在一般情况下的张口，不会伴随出现大阳线或大阴线，张口程度相对有限，而这是短期情绪的小爆发，但不极端，随后一般伴随一小段涨幅或跌幅，而走势不会太快。

一般布林通道张口后，可以看作是上涨或下跌趋势打开的讯号。

# 三轨收口

布林通报一般情况的三轨收口出现时，盘面容易进入盘整，故基本上，只要布林通道维持收口形态，就得持续等待新的方向出现，等张口或是等后续出现三轨向上移动或三轨向下移动，这就是给新的短期方向选择。

# 张口型喇叭口

布林通道的「张口型喇叭口」一向为交易者所重视，因此种型态出现时并不好做交易，因对应到的可能是极端行情。

布林通道的「张口型喇叭口」出现时，通常是短期市场情绪急剧爆发，并伴随盘面大幅波动，但布林通道「张口型喇叭口」型态无法在行情启动前发出信号，也就是说，等「张口型喇叭口」出现时，盘面行情可能已经走掉大半部分，这对交易者来说讯号出现时常常已经太晚。

# 收口型喇叭口

「收口型喇叭口」常常紧跟出现在「张口型喇叭口」之后，这是对应极端行情出现后的修正阶段，本质无异于上述收口形态，亦不建议操作。

# 紧口型喇叭口

此种型态较为少见，是本来预示盘整的「三轨水准运行」的布林通道，却突然收窄宽度，向弹簧一样瞬间压缩，为后续情绪爆发做准备，但在何时、何位置爆发较难掌握。

整体来说，将布林通道的张口或收口作为市场情绪波动的信号，能确认行情打开，以此来辅助入场。

# 缺点与技术限制

任何一项指标皆有缺点以及技术限制，而布林通道的缺点即在于，如果使用的均线周期过短、且价格短期波动剧烈的话，那么通道的带宽变化即会很大。

布林通道的技术限制则在于，如果当日开盘出现跳空大涨或大跌之市况，即会造成通道的带宽突然放大，故而使得这段期间的通道丧失参考价值，这属于布林通道上的先天限制。

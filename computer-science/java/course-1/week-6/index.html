# еӯ—з¬Ұзұ»еһӢ char

  • еӯ—з¬Ұд№ҹжҳҜ Java дёӯеҹәзЎҖзҡ„ж•°жҚ®зұ»еһӢд№ӢдёҖ
    • з”ЁеҚ•еј•еҸ·иЎЁзӨәзҡ„еӯ—з¬Ұпјҡ 'a' , '1'
  • Java йҮҮз”Ё Unicode16 иЎЁиҫҫеӯ—з¬ҰпјҢеңЁжүҖжңүзҡ„жңәеҷЁдёҠпјҢдёҚз®Ў CPUгҖҒж“ҚдҪңзі»з»ҹе’Ңжң¬ең°иҜӯиЁҖпјҢеӯ—з¬Ұзұ»еһӢжҳҜдёҖиҮҙе’Ңз»ҹдёҖзҡ„гҖӮ
  • дёҖдёӘжұүеӯ—д№ҹжҳҜ Unicode зҡ„дёҖдёӘеӯ—з¬ҰпјҢжүҖд»Ҙд№ҹжҳҜ Java зҡ„дёҖдёӘеӯ—з¬ҰгҖӮ

# еӯ—з¬Ұи®Ўз®—

  • дёҖдёӘеӯ—з¬ҰеҠ дёҖдёӘж•°еӯ—еҫ—еҲ° Unicode з ҒиЎЁдёӯзҡ„йӮЈдёӘж•°д№ӢеҗҺзҡ„еӯ—з¬Ұ
  • дёӨдёӘеӯ—з¬Ұзҡ„еҮҸпјҢеҫ—еҲ°е®ғд»¬еңЁиЎЁдёӯзҡ„и·қзҰ»
  • char д№ҹеҸҜд»Ҙе’Ң int д№Ӣй—ҙзӣёдә’иөӢеҖјпјҢиҫ“еҮәеӯ—з¬ҰеңЁ Unicode з ҒиЎЁдёӯзҡ„дҪҚзҪ®

# еӨ§е°ҸеҶҷиҪ¬жҚў

  • еӯ—жҜҚе’Ңж•°еӯ—еңЁ Unicode иЎЁдёӯжҳҜйЎәеәҸжҺ’еҲ—зҡ„
  • еӨ§еҶҷеӯ—жҜҚе’Ңе°ҸеҶҷеӯ—жҜҚжҳҜеҲҶејҖжҺ’еҲ—зҡ„пјҢдёҚеңЁдёҖиө·
  • 'a' - 'A' еҸҜд»Ҙеҫ—еҲ°дёӨж®өд№Ӣй—ҙзҡ„и·қзҰ»
  • a + 'a' - 'A' еҸҜд»ҘжҠҠдёҖдёӘеӨ§еҶҷеӯ—жҜҚеҸҳжҲҗе°ҸеҶҷеӯ—жҜҚ
  • a + 'A' - 'a' еҸҜд»ҘжҠҠдёҖдёӘе°ҸеҶҷеӯ—жҜҚеҸҳжҲҗеӨ§еҶҷеӯ—жҜҚ

# еӯ—з¬ҰеӨ§е°ҸжҜ”иҫғ

  • еӯ—з¬ҰеҸҜд»ҘжҜ”иҫғеӨ§е°ҸпјҢдҫқжҚ®жҳҜе®ғд»¬еңЁ Unicode иЎЁдёӯзҡ„зј–еҸ·
